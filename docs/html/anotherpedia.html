Anotherpedia

Anotherpedia is a knock-off of Wikipedia, being an online encyclopedia website full of different Fact and useless knowledge about Thing.

Anotherpedia was made by Warm_Wooly on August 28th, 2023, making Anotherpedia ¤tAge day old. The first page created was first. Since its creation, Anotherpedia has amassed a total of &allPages page (Anotherpedia), beating its goal of making 1000 pages on July 10th, 2024. The next goal is 2500 pages.

The logo, comprised of three short cylinder using the grayish-blue the webpage's color scheme, was made in Google Drawings. There is no symbolism nor meaning behind the logo. On May 31st, 2025, to avoid Shutdown of Glitch, Anotherpedia was then moved to GitHub Pages. On March 26th, 2025, anotherpedia.com was registered on Namecheap, redirecting to anotherpedia.glitch.me. Following the move to GitHub Pages, the domain anotherpedia.com would be used for Anotherpedia, hosted on warmwooly.github.io/Anotherpedia/. Anotherpedia uses an HTML file called index.html. It is linked to a CSS file titled style.css, and a pair of JavaScript files names script.js and pages.js. script.js contains all the code for website display, page selection, tooltip, text generation, and other User Experience-related tasks, while pages.js holds all the data for the pages and redirects, including this one!

image popup (anotherpedia) are controlled by popup.html (to handle the window for displaying the image) and popup.js (to handle popup transformations). Pages are stored as website snapshot in a folder, which contains the raw contents of the page to be scraped by web crawler. There is also a sitemap, sitemap.xml, that lists all the available pages.

Anotherpedia uses Cloudflare to get Discord webhook to send a text file to the Anotherpedia Discord, which is protected with reCAPTCHA. Cloudflare is also used to redirect from snapshots and to grab the latest pages.js. Anotherpedia had a sister website, Anotherpedia.txt, which was used to store and share text and pages. Anotherpedia is a portmanteau of another and Wikipedia. Anotherpedia used to go under the name Stupipedia.
