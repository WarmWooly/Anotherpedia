Anotherpedia

Anotherpedia is a knock-off of Wikipedia, being an encyclopedic website full of different facts and useless knowledge about things.

Anotherpedia was made by Warm_Wooly on August 28th, 2023, making Anotherpedia ¤tAge days old. The first page created was first. Since its creation, Anotherpedia has amassed a total of &allPages pages, beating its goal of making 1000 pages on July 10th, 2024. The next goal is 2500 pages.

The logo, comprised of three short cylinders using the grayish-blue the webpage's color scheme, was made in Google Drawings. There is no symbolism nor meaning behind the logo. It was updated on February 24, 2024, to now have a border and an increase in image quality, alongside converting it from .ico to .svg. The mascot for Anotherpedia is Anno the flow worm.

Anotherpedia uses an HTML file called index.html for its main source. It is linked to a CSS file titled style.css, and a pair of JavaScript files names script.js and pages.js. script.js contains all the code for website display, page selection, tooltips, text generation, and other UX-related tasks, while pages.js holds all the data for the pages and redirects, including this one! A separate theme.js file also holds the themes and other preset Settings options.

Image popups are controlled by popup.html (to handle the window for displaying the image) and popup.js (to handle popup transformations). Pages are stored as snapshots in a folder, which contains the raw contents of the page to be scraped by web crawlers (for example, this page is anotherpedia.html for crawlers). There is also a sitemap, sitemap.xml, that lists all the available pages.

Anotherpedia uses Cloudflare to get Discord webhooks to send a text file to the Anotherpedia Discord, which is protected with reCAPTCHA. Cloudflare is also used to redirect from snapshots and to grab the latest pages.js. Anotherpedia had a sister website, Anotherpedia.txt, which was used to store and share text and pages. Anotherpedia is a portmanteau of another and Wikipedia. Anotherpedia used to go under the name Stupipedia.
