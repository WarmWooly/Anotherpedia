Anotherpedia log

<>This is the {{bAnotherpedia log}}, acting as a crude [[update log]] for [[Anotherpedia]], with the most recent changes being shown at the top. The solution is straightforward: giving a 15-second timeout per submission. This will mainly show that the page has been successfully sent to prevent user error. This isn't nearly close to anything robust, but it is good for now. If the case arises, additional restrictions (such as enabling challenges through reCAPTCHA per submission) can be put into place, but this system does the bare minimum while being (hopefully) bug-free.<><><>[[Infoboxes|infobox]] have been added to Anotherpedia! With this comes the removal (through a multi-hour editing session) of those clunky "Stats" sections. Instead, there will now be a box that can contain images and information about a topic! While it's scope is primarily focused towards simple information from pages that already had an pre-infobox workaround, infoboxes may be used in more types of pages in the future.<><>Pages can now be subitted to the Anotherpedia [[Discord|Discord (application)]] (which can be joined at <>) with .txt files! While this solution isn't as robust as Anotherpedia.txt, it is a good solution in the meantime that fixes the issue of non-members from being unable to submit pages, alongside making it easier for page creation and edit requests to be made. I'd like to revisit this in the future to more closely align with a 1-link solution to view new page requests, but this is a huge step in that direction.
