Anotherpedia speedrun

An {{bAnotherpedia speedrun}} is Anotherpedia clone of Wikiracing, where the player attempts to speedrun from a start page (Anotherpedia) to an end page. These are the official rules for Anotherpedia speedruns:&spThe run starts when the starting link is clicked.&spA winner will be determined by the number of pages or backs the players use or the time it takes to reach the end page.&spOnly the link (Anotherpedia) on the page and the back button may be used for page navigation.&spNo use of Ctrl + F or other similar features on the web browser.&sp(Optional) No tooltips.
