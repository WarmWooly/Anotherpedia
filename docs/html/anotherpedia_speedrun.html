Anotherpedia speedrun

An Anotherpedia speedrun is Anotherpedia's clone of Wikiracing, where the player attempts to speedrun from a starting page to an ending page. These are the official rules for Anotherpedia speedruns:
The run starts when the starting link is clicked.
A winner will be determined by the number of pages or backs the players use or the time it takes to reach the end page.
Only the links on the page and the back button may be used for page navigation.
No use of Ctrl + F or other similar features on the browser.
(Optional) No tooltips. Why not try it yourself?
