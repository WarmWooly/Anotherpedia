Aquadynamic Leather Padding (Not Lazy Chainmail)

<>Aqyadynamic leather padding is an item in the Minecraft mod Not Lazy Chainmail as an upgrade to the leather padding. Aquadynamic leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving +0.33 Swim Speed. Aquadynamic leather padding can be melded in a Melder using 1 leather padding and 2 prismarine shards. In a leathersmithing table, aquadynamic leather padding can be applied to chainmail armor or shields.
