Array (computer programming)

An array is a variable used in multiple programming languages to store a large amount of values. The only limit to an array is the computer's allocated memory. const array = [1, 2, 3, 4, 5]
console.log(array[2]) // Prints '3'
console.log(array[4]) // Prints '5' int array[] = {1, 2, 3, 4, 5}
printf("%d", array[0]); // Prints '1'
printf("%d", array[3]); // Prints '4' array = {1, 2, 3, 4, 5}
print(array[1]) -- Prints '1'
print(array[5]) -- Prints '5' @array = (1, 2, 3, 4, 5);
print "$array[2]\n"; ## Prints '3'
print "$array[4]\n"; ## Prints '5'
