Automated Permanence (Not Lazy Chainmail)

<>{{bAutomated Permanence}} is an enchantment (Minecraft) in the {{iMinecraft}} Minecraft modding {{iNot Lazy Chainmail}} applied to industrial belts (Not Lazy Chainmail), sieve meshes (Not Lazy Chainmail), and pliers (Not Lazy Chainmail), to make them unbreakable. Industrial belts, sieve meshes, and pliers can have Automated Permanence applied by the automated permanence rune (Not Lazy Chainmail). To cast the rune, the user must have 2 experience (Minecraft), with the rune destroyed on use. Items with Automated Permanence will not lose durability (Minecraft) on use.<> This includes manual operations (i.e. snipping (Not Lazy Chainmail) with pliers) and items used within various machines. Automated Permanence allows items used in machines not to break, preventing automated constructions using {{iNot Lazy Chainmail}} from requiring the player to constantly resupply machines with new tools.
