<><>||{{bMaximum level}}|I||{{bPrimary items}}|[[Industrial belt|Industrial Belts (Not Lazy Chainmail)]], [[sieve mesh|sieve meshes (Not Lazy Chainmail)]], [[pliers|pliers (Not Lazy Chainmail)]]||{{bIncompatible with}}|[[Separation|Separation (Not Lazy Chainmail)]]info>>{{bAutomated Permanence}} is an [[enchantment|enchantment (Minecraft)]] in the {{i[[Minecraft]]}} [[mod|Minecraft modding]] {{i[[Not Lazy Chainmail]]}} applied to [[industrial belts|industrial belts (Not Lazy Chainmail)]], [[sieve meshes|sieve meshes (Not Lazy Chainmail)]], and [[pliers|pliers (Not Lazy Chainmail)]], to make them unbreakable.<>Industrial belts, sieve meshes, and pliers can have Automated Permanence applied by the [[automated permanence rune|automated permanence rune (Not Lazy Chainmail)]]. To cast the rune, the user must have 2 [[levels|experience (Minecraft)]], with the rune destroyed on use.<>Items with Automated Permanence will not lose [[durability|durability (Minecraft)]] on use.<> This includes manual operations (i.e. [[snipping|snipping (Not Lazy Chainmail)]] with pliers) and items used within various machines. Automated Permanence allows items used in machines not to break, preventing automated constructions using {{iNot Lazy Chainmail}} from requiring the player to constantly resupply machines with new tools.
