Automated Permanence (Not Lazy Chainmail)

<>Automated Permanence is an enchantment in the Minecraft mod Not Lazy Chainmail applied to industrial belts, sieve meshes, and pliers, to make them unbreakable. Industrial belts, sieve meshes, and pliers can have Automated Permanence applied by the automated permanence rune. To cast the rune, the user must have 2 levels, with the rune destroyed on use. Items with Automated Permanence will not lose durability on use. This includes manual operations (i.e. snipping with pliers) and items used within various machines. Automated Permanence allows items used in machines not to break, preventing automated constructions using Not Lazy Chainmail from requiring the player to constantly resupply machines with new tools.
