Automated Permanence Rune ({{iNot Lazy Chainmail}})

<><>||{{bStats}}||{{b[[Rarity|Rarity (Minecraft)]]}}|Common||{{b[[Stack size|Stack size (Minecraft)]]}}|64||{{b[[Fire/lava&spresistance|Fire-resistant item (Minecraft)]]}}|Noinfo>>The {{bautomated permanence rune}} is a [[rune|runes (Not Lazy Chainmail)]] in the {{i[[Minecraft]]}} [[mod|mod (Minecraft)]] {{i[[Not Lazy Chainmail]]}}. It is used to apply [[automated permanence|automated permanence (Not Lazy Chainmail)]] to [[industrial belts|industrial belts (Not Lazy Chainmail)]], [[sieve meshes|sieve meshes (Not Lazy Chainmail)]], and [[pliers|pliers (Not Lazy Chainmail)]].<><>The automated permanence rune can be melded in a [[Melder|Melder (Not Lazy Chainmail)]] using 1 [[rose gold ingot|rose gold ingot (Not Lazy Chainmail)]] and 1 [[pseudorock|pseudorock (Not Lazy Chainmail)]].<><>The automated permanece rune can be cast onto an [[industrial belt|industrial belts (Not Lazy Chainmail)]], [[sieve mesh|sieve meshes (Not Lazy Chainmail)]], or [[pliers|pliers (Not Lazy Chainmail)]], using 2 [[levels|experience (Minecraft)]] and destroying the rune. This is performed by holding the rune in the [[offhand|offhand (Minecraft)]] and the item to cast in the [[main hand|main hand (Minecraft)]], then using the rune with the required levels, applying [[Automated Permanence|automated permanence (Not Lazy Chainmail)]]. The automated permanence rune will not cast if the item is already enchanted with Automated Permanence or if the item is enchanted with [[Separation|Separation (Not Lazy Chainmail)]]. Upon using an automated permanence rune, the [[advancement|advancement (Minecraft)]] [[Unbreaking M|Unbreaking M (Not Lazy Chainmail)]] will be awarded.
