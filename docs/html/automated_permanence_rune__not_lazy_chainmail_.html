It is used to apply automated permanence (Not Lazy Chainmail) to industrial belts (Not Lazy Chainmail), sieve meshes (Not Lazy Chainmail), and pliers (Not Lazy Chainmail). The automated permanence rune can be melded in a Melder (Not Lazy Chainmail) using 1 rose gold ingot (Not Lazy Chainmail) and 1 pseudorock (Not Lazy Chainmail). The automated permanece rune can be cast onto an industrial belts (Not Lazy Chainmail), sieve meshes (Not Lazy Chainmail), or pliers (Not Lazy Chainmail), using 2 experience (Minecraft) and destroying the rune. This is performed by holding the rune in the offhand (Minecraft) and the item to cast in the main hand (Minecraft), then using the rune with the required levels, applying automated permanence (Not Lazy Chainmail). The automated permanence rune will not cast if the item is already enchanted with Automated Permanence or if the item is enchanted with Separation (Not Lazy Chainmail). Upon using an automated permanence rune, the advancement (Minecraft) Unbreaking M (Not Lazy Chainmail) will be awarded.
