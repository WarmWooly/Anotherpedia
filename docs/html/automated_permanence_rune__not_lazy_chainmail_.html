Automated Permanence Rune (Not Lazy Chainmail)

<>The automated permanence rune is a rune in the Minecraft mod Not Lazy Chainmail. It is used to apply automated permanence to industrial belts, sieve meshes, and pliers. The automated permanence rune can be melded in a Melder using 1 rose gold ingot and 1 pseudorock. The automated permanece rune can be cast onto an industrial belt, sieve mesh, or pliers, using 2 levels and destroying the rune. This is performed by holding the rune in the offhand and the item to cast in the main hand, then using the rune with the required levels, applying Automated Permanence. The automated permanence rune will not cast if the item is already enchanted with Automated Permanence or if the item is enchanted with Separation. Upon using an automated permanence rune, the advancement Unbreaking M will be awarded.
