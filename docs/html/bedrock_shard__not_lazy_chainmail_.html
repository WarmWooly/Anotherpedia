Bedrock Shard (Not Lazy Chainmail)

<>{{bBedrock shards}} are an item (Minecraft) in the {{iMinecraft}} mod (Minecraft) {{iNot Lazy Chainmail}} used for creating iron composite ingot (Not Lazy Chainmail) and pseduorock (Not Lazy Chainmail). When loose bedrock (Not Lazy Chainmail) is mining (Minecraft) with a diamond pickaxe (Minecraft) or better, it will drop 1 bedrock shard. Fortune (Minecraft) does not affect shard drops. Loose bedrock can be found below y-level (Minecraft) -54, replacing bedrock (Minecraft). Loose bedrock can be obtained when a Crusher (Not Lazy Chainmail) crushes pseudorock (Not Lazy Chainmail) (makes 3 shards). When temporal sand (Not Lazy Chainmail) is sieving (Not Lazy Chainmail) in an iron composite sieve mesh (Not Lazy Chainmail), there is a 1% chance to drop a bedrock shard. Separation (Not Lazy Chainmail) can increase the chance/drops of bedrock shards from sieving.
