Bits & Bops

{{iBits & Bops}} is an indie rhythm game released on December 9, 2025, by the Australian game studio Tempo Lab Games. It features gameplay similar to Rhythm Heaven, comprised of different minigames where the player must time their input to the action on-screen to the music. It additionally has mixtapes, where multiple minigames are spliced together in a song. The game features a large cast that appear throughout the main game and in minigames.

The game was directed and designed by Evan Andrews, with writing, art, and additional designing by Rose Hammer. The main composer is Julian Sanchez (Zorsy), followed by Lich, with sound design by Matthew Chin. Other animators include Jacinta Joe, Nirali Somaia, and Joel Williams, with promotional art by Kevin Fagaragan. For additional credits, see Bits & Bops credits. The website for the game is at <>.
