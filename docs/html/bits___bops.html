Bits & Bops

{{iBits & Bops}} is a rhythm game released on December 9, 2026, by the Australian game studio Tempo Lab Games. It features gameplay similar to Rhythm Heaven, comprised of different minigames where the player must time their input to the action on-screen. It additionally has mixtapes, where multiple minigames are spliced together in a song.
