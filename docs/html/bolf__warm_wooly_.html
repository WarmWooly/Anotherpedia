Bolf (Warm_Wooly)

{{iBolf}} is a top-down 2D golf game by Warm_Wooly made in GameMaker, first being available on October 2021 on itch.io. Bolf can use mouse- or keyboard-only controls and runs on Windows. Bolf uses a randomly generated arrangement of tiles to create a course, where the player must shoot the ball into the hole within 5 strokes. As the game progresses, more randomized tiles and items are added to the board. The goal is to see how long the player can last before getting a bogey (going over the allotted strokes). Bolf is a portmanteau of bad and golf. In Bolf, there are 7 tiles and 3 items in total: sand, ice, water, twist, boost, conveyor, pain, stroke refill, stroke deduction, and stroke mystery. The board is 11x10 (width x height), with the ball spawning on the position (5, 10). The hole can be located anywhere besides where the ball spawns.
