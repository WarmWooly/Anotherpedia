Bottom text

Bottom text refers to a subversion of the top text bottom text image macro for internet meme, where instead of populating the bottom field with the punchline or other joke, it is only "bottom text". It can be used to make the top text be read as deadpan, as if it is a normal sentence or statement. It can also be used to poke fun at meme generator that template with "top text" and "bottom text" by deliberately leaving the meme incomplete. It can additionally have the top text unfilled, or in rare cases, just the top text incomplete. Bottom text is often paired with reaction image.
