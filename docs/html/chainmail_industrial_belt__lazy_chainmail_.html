Chainmail Industrial Belt (Not Lazy Chainmail)

Not to be confused with Iron Chainmail Industrial Belt ({{aiNot Lazy Chainmail).}}
<>The chainmail industrial belt is an item (Minecraft) in the Minecraft mod (Minecraft) Lazy Chainmail to run the Winding Jig (Lazy Chainmail), and an upgrade to the kelp industrail belt (Lazy Chainmail) or leather industrial belt (Lazy Chainmail). Chainmail industrial belts can be created through Chainmail project (Lazy Chainmail) with the setting "Industrial Belt". The chainmail industrial belt takes 256 iron chain link (Lazy Chainmail) to craft. The chainmail industrial belt can be used to run the Winding Jig, using a durability every time an iron wire (Lazy Chainmail) or iron spool (Lazy Chainmail) is fully wound. Unbreaking (Minecraft) decreases the chance that it takes damage, having a 50%, 33%, and 25% chance for levels I, II, and III respectively. The chainmail industrial belt is three times faster than the crafting (Minecraft) industrial belts. Chainmail industrial belts can be repaired through combining (Minecraft) by crafting (Minecraft), using a grindstone (Minecraft), or using an anvil (Minecraft). Combining chainmail industrial belts will add the two durability values together with an additional 3 durability. Combining will remove all enchantments, with some experience (Minecraft) returned when combining in a grindstone. Combining in an anvil will keep and combine the enchantments on the industrial belts, costing experience. Chainmail industrial belts can be enchanting (Minecraft) with Unbreaking (Minecraft), Mending (Minecraft), and Curse of Vanishing (Minecraft) using an enchanted book (Minecraft) on an anvil (Minecraft). Chainmail industrial belts cannot be enchanted in an enchanting table.
