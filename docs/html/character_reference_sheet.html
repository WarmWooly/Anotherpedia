Character reference sheet

A character reference sheet or character model sheet is a reference tool for how to draw or otherwise illustrate a character and/or important objects. The primary purpose of a character reference sheet is to keep a character on-model and consistent between different drawings and artists. Character reference sheets tend to have the character in different poses, angles, and expressions. They may also have a color palette, relevant tools or important objects, close-ups on detailed elements, other forms of the character, or important character information.
