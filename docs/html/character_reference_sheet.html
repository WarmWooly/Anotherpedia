Character reference sheet

A character reference sheet or character model sheet is a reference tool for how to drawing or otherwise illustration a character and/or important object. The primary purpose of a character reference sheet is to keep a character on-model and character consistency between different drawings and artist. Character reference sheets tend to have the character in different pose, perspective (graphical), and facial expression. They may also have a color palette, relevant tool or important objects, close-ups on detailed elements, other forms of the character, or important character information.
