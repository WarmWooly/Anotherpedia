Chromatic Mass (Not Lazy Chainmail)

<resistance|Noinfo>>The chromatic mass is an item in the Minecraft mod Not Lazy Chainmail that needs to be processed into chromatic dust for use. Chromatic mass can be melded in a Melder using 1 diamond, 1 emerald, and 1 amethyst shard. Pseudorock can be crushed into 1 chromatic dust. Chromatic mass can be used in a smithing table to trim armor.
