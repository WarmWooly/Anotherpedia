Chubik Remade

<><>{{bChubik Remade}} is a {{i[[Minecraft]]}} [[texture pack|Minecraft texture pack]] created by iplaygamesbad_ posted onto [[CurseForge]] on July 24, 2022, to update the [[Chubik texture pack|Chubik (texture pack)]] to modern versions.<> Unlike the original pack, only [[blocks|block (Minecraft)]] are updated, with [[mobs|mob (Minecraft)]] and [[items|item (Minecraft)]] left untouched.<> Additionally, [[water|water (Minecraft)]] is [[opaque]] instead of [[transparent]], textures prefer the file name over the block's name (such as Grass Block Overlay instead of [[Grass Block|Grass Block (Minecraft)]]), and each texture is 256x256 in size. While it was originally made for [[1.19|Minecraft: Java Edition 1.19]], <>, it's available for versions up to [[1.21.3|Minecraft: Java Edition 1.21.3]].

This is a pre-rendered snapshot for search engines.
