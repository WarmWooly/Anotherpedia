Chubik Remade

Chubik Remade is a Minecraft Minecraft texture pack created by iplaygamesbad_ posted onto CurseForge on July 24, 2022, to update the Chubik (texture pack) to modern versions. Unlike the original pack, only block (Minecraft) are updated, with mob (Minecraft) and item (Minecraft) left untouched. Additionally, water (Minecraft) is opaque instead of transparent, textures prefer the file name over the block's name (such as Grass Block Overlay instead of Grass Block (Minecraft)), and each texture is 256x256 in size. While it was originally made for Minecraft: Java Edition 1.19, <>, it's available for versions up to Minecraft: Java Edition 1.21.3.
