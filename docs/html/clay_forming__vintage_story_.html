Clay forming (Vintage Story)

Clay forming is a crafting process within Vintage Story to create various pottery using clay. Formed clay can be smelted in a pit kiln or a beehive kiln. After selecting a recipe, clay voxels can be added and removed from the craft until it has been shaped to match the guidelines given. Only one layer can be worked on at a time, with a 1x1, 2x2, 3x3, and layer duplication tools to speed up the process. The following can be molded using blue, red, or fire clay:
