Clay forming ({{iVintage Story}})

<><>{{bClay forming}} is a [[crafting process|crafting process (Vintage Story)]] within {{i[[Vintage Story]]}} to create various [[pottery|pottery (Vintage Story)]] using [[clay|clay (Vintage Story)]]. Formed clay can be smelted in a [[pit kiln|pit kiln (Vintage Story)]] or a [[beehive kiln|beehive kiln (Vintage Story)]]. After selecting a recipe, clay [[voxels|voxel]] can be added and removed from the craft until it has been shaped to match the guidelines given.
