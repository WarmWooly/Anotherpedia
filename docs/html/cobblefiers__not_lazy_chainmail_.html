Cobblefiers (Not Lazy Chainmail)

For the specific item, see Cobblefier ({{aiNot Lazy Chainmail).}}&sp<>Cobblefiers are a type of item in the Minecraft mod Not Lazy Chainmail. Cobblefiers are used in the Cobblestone Generator to modify its output. All cobblefiers are crafted, all requiring the cobblefier as a base. Some cobblefiers require items, while others are a mix between two or more cobblefiers. Cobblefiers can be placed into a Cobblestone Generator to create various materials, with different cobblefiers running faster/slower depending on the value of the item. Some items (such as soul sand) that can be generated are much slower to compensate for their use in other processes (i.e. sieving). When all the 52 cobblefiers are placed in an Expanded Barrel Collector, the player will be awarded the advancement Cobblestone is Everything.
