Constructive Leather Padding (Not Lazy Chainmail)

<>Constructive leather padding is an item in the Minecraft mod Not Lazy Chainmail as an upgrade to the leather padding. Constructive leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving +1 Block Reach. Constructive leather padding can be melded in a Melder using 1 leather padding and 1 chiseled stone brick. In a leathersmithing table, constructive leather padding can be applied to chainmail armor or shields. Each constructive leather padding adds +1 Block Reach, with a full set of armor with 12 constructive leather paddings giving +12 Block Reach. Shields will only add Block Reach when held in the main hand or offhand.
