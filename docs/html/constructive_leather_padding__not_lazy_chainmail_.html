Constructive Leather Padding (Not Lazy Chainmail)

<>Constructive leather padding is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the leather padding (Not Lazy Chainmail). Constructive leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +1 Block Reach (Forge). Constructive leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 1 chiseled stone bricks (Minecraft). In a leathersmithing table (Not Lazy Chainmail), constructive leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each constructive leather padding adds +1 Block Reach (Forge), with a full set of armor with 12 constructive leather paddings giving +12 Block Reach. Shields will only add Block Reach when held in the main hand (Minecraft) or offhand (Minecraft).
