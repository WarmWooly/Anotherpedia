Container

<><>A {{bcontainer}} is any [[object]] which holds, or [[contains|contain]], another object. The [[size]] of the container determines how much it can contain, known as the container's [[volume]]. Containers are typically used to [[transport|transportation]] [[bulk goods]], [[loose materials|loose material]], [[liquids|liquid]], and [[gasses|gas]]. Containers can be contained in containers and contain containers.
