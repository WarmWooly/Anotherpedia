Continuous flow through vermicomposter

A continuous flow through vermicomposter or CFT vermicomposter is a vermicomposter in which compostable waste is placed at the top to be broken down (primarily using worms), which slowly trickles down as usable compost. The main upside of a CFT vermicomposter is the ability to feed in new stock while allowing the current ecosystem above to remain unharmed, only requiring periodic watering and a system to knock off new compost. However, they tend to be much larger systems compared to other methods of composting which require more maintenance.
