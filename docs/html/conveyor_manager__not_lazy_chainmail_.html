Conveyor Manager (Not Lazy Chainmail)

The conveyor manager is a block in the Minecraft mod Not Lazy Chainmail used to redirect and split items on chain conveyors. The conveyor manager can be crafted with 4 rose gold ingots, 4 spool blocks, and 1 crude engine (place the engine in the middle, the spool blocks on the sides, and the rose gold in the corners; makes 4). The conveyor manager will detect chain conveyors in any of the cardinal directions from the block. Right-clicking the conveyor manager will open up its menu. Each direction can be set to Disabled for no action, Input for items to be taken from, and Output for items to be placed to. If there are no Outputs, no items will be taken from Inputs. Valid Inputs (if a conveyor is adjacent to the side) will have an orange particle effect, and valid Outputs will have a green particle effect. If there is an Input or Output assigned but no valid conveyor, a white particle effect will be displayed. Particles can be disabled using the "Particles" button.

Each item stack will be moved to a valid Output. If there is only one Output, all items will exit from it. Multiple Outputs has each item stack exit to the next valid Output, rotating clockwise.
