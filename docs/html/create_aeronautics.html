Create Aeronautics

>
{{iCreate Aeronautics}} is a large Minecraft mod (Minecraft) as an minecraft mod addon to Create (Minecraft mod), made by the Simulated Team and released on April 19, 2026. It adds the ability to turn block (Minecraft) into rigid body, which are physics simulation, while adding new Create Aeronautics blocks and Create Aeronautics items to create and interact with these rigid bodies. It builds off of Create's rotational force (Create) system to power blocks that can generate propulsion (Create Aeronautics) or roll along surfaces. Gravity (Create Aeronautics), lift (Create Aeronautics), and drag (Create Aeronautics) are simulated alongside balloon lift (Create Aeronautics), levitation (Create Aeronautics), and magnetic (Create Aeronautics) as additional forces that can be imparted on a simulated contraption (Create Aeronautics). Vehicles such as car (Create Aeronautics), hot air balloon (Create Aeronautics), and plane (Create Aeronautics) can be made utilizing dedicated blocks like wheel mount (Create Aeronautics) and hot air envelope (Create Aeronautics). Create Aeronautics is available for Minecraft: Java Edition 1.21.1 for NeoForge. The mod itself is technically a combination of Create Simulated (primarily handling physics interations), Create Aeronautics (blocks to create flying contraptions), and Create Offroad (blocks to create car-like contraptions), but is most often used and described as a single mod. Create Aeronautics requires Create and the library mod Sable (Minecraft mod).
