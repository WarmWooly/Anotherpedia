Create Aeronautics

>
{{iCreate Aeronautics}} is a large Minecraft mod as an addon to Create, made by the Simulated Team and released on April 19, 2026. It adds the ability to turn blocks into rigid bodies, which are simulated, while adding new blocks and items to create and interact with these rigid bodies. It builds off of Create's rotational force system to power blocks that can generate propulsion or roll along surfaces. Gravity, lift, and drag are simulated alongside balloon lift, levitation, and magnetism as additional forces that can be imparted on a simulated contraption. Vehicles such as cars, hot air balloons, and planes can be made utilizing dedicated blocks like wheel mounts and hot air envelopes. Create Aeronautics is available for 1.21.1 for NeoForge. The mod itself is technically a combination of Create Simulated (primarily handling physics interations), Create Aeronautics (blocks to create flying contraptions), and Create Offroad (blocks to create car-like contraptions), but is most often used and described as a single mod. Create Aeronautics requires Create and the library Sable.
