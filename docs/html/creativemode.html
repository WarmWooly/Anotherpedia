CreativeMode

CreativeMode (<>) is a generative artificial intelligence Minecraft Minecraft modding creator, founding in 2024 and released in 2025. The service can create mods for block (Minecraft), item (Minecraft), mob (Minecraft), and structure (Minecraft) <>, with biome (Minecraft) and complete mod packages "coming soon". It can create the mechanics and texture (Minecraft) with code generation and image generation from a text prompt. The company was founded by Wilson Spearman, Jeffrey Tsaw, and Coleman Smith (CreativeMode founder).
