Crush'n'crush'n'smelt'n'tintifier (Not Lazy Chainmail)

<resistance|Noinfo>>The crush'n'crush'smelt'n'tintifier is an item in the Minecraft mod Not Lazy Chainmail used as a modifier for the Cobblestone Generator to produce tinted glass. Crush'n'crush'n'smelt'n'tintifiers can be crafted using 1 crush'n'crush'n'smeltifier and 8 amethyst shards (surround the crush'n'crush'n'smeltifier with the shards). Crush'n'crush'n'smelt'n'tintifiers, when put in a Cobblestone Generator, will create tinted glass. Crush'n'crush'n'smelt'n'tintifiers take 128x as long to generate one tinted glass compared to one cobblestone. When placed in an Expanded Barrel Collector alongside the 51 other cobblefiers, the player will be awarded the advancement Cobblestone is Everything.
