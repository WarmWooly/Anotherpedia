Crush'n'crush'n'smelt'n'tintifier (Not Lazy Chainmail)

<>The crush'n'crush'smelt'n'tintifier is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail used as a modifier for the Cobblestone Generator (Not Lazy Chainmail) to produce tinted glass (Minecraft). Crush'n'crush'n'smelt'n'tintifiers can be crafting (Minecraft) using 1 crush'n'crush'n'smeltifier (Not Lazy Chainmail) and 8 amethyst shard (Minecraft) (surround the crush'n'crush'n'smeltifier with the shards). Crush'n'crush'n'smelt'n'tintifiers, when put in a Cobblestone Generator, will create tinted glass (Minecraft). Crush'n'crush'n'smelt'n'tintifiers take 128x as long to generate one tinted glass compared to one cobblestone (Minecraft). When placed in an Expanded Barrel Collector (Not Lazy Chainmail) alongside the 51 other cobblefiers (Not Lazy Chainmail), the player will be awarded the advancement (Minecraft) Cobblestone is Everything (Not Lazy Chainmail).
