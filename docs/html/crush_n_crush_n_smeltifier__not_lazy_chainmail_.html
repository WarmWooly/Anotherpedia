Crush'n'crush'n'smeltifier (Not Lazy Chainmail)

<>The crush'n'crush'smeltifier is an item in the Minecraft mod Not Lazy Chainmail used as a modifier for the Cobblestone Generator to produce glass. Crush'n'crush'n'smeltifiers can be crafted using 1 crush'n'crushifier and 1 smeltifier (shapeless). Crush'n'crush'n'smeltifiers are used in the following crafting recipes:&spCrush'n'crush'n'smelt'n'tintifier - 1 crush'n'crush'n'redifier and 8 amethyst shards (surround the crush'n'crush'n'smeltifier with the shards). Crush'n'crush'n'smeltifiers, when put in a Cobblestone Generator, will create glass. Crush'n'crush'n'smeltifiers take 32x as long to generate one glass compared to one cobblestone. When placed in an Expanded Barrel Collector alongside the 51 other cobblefiers, the player will be awarded the advancement Cobblestone is Everything.
