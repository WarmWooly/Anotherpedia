Crystalized instance (SCP: Devolved Revamp)

A crystalized instance is a lethal NPC created by SCP-409 in SCP: Devolved Revamp. A crystalized instance is created whenever a player is killed by SCP-409, spawning on top of their body. Upon contact, a player will become infected and take 5 damage/second. On death, the player will create another crystalized instance.
