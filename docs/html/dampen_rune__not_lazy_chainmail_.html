Dampen Rune (Not Lazy Chainmail)

The dampen rune is a rune in the Minecraft mod Not Lazy Chainmail. It is used to apply Dampened to Not Lazy Chainmail tools. The dampen rune can be melded in a Melder using 1 machine dampener and 1 smooth stone. The dampen rune can be cast onto a chainsaw, chainsword, chain launcher, chainmail project, sieve mesh, pliers, rune scraper, whip, flail, nunchucks, or lasso, using 1 level. This is performed by holding the rune in the offhand and the item to cast in the main hand, then using the rune with the required levels, applying Dampened. The dampened rune will not cast if the item is already enchanted with Dampened or if the item is enchanted with Muffled.
