Destructive Leather Padding (Not Lazy Chainmail)

<>Destructive leather padding is an item in the Minecraft mod Not Lazy Chainmail as an upgrade to the leather padding. Destructive leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving +1 Entity Reach. Destructive leather padding can be melded in a Melder using 1 leather padding and 1 cracked stone brick. In a leathersmithing table, destructive leather padding can be applied to chainmail armor or shields. Each destructive leather padding adds +1 Entity Reach, with a full set of armor with 12 destructive leather paddings giving +12 Entity Reach. Shields will only add Entity Reach when held in the main hand or offhand.
