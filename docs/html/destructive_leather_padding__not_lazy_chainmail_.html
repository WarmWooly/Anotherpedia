Destructive Leather Padding (Not Lazy Chainmail)

<>{{bDestructive leather padding}} is an item (Minecraft) in the {{iMinecraft}} mod (Minecraft) {{iNot Lazy Chainmail}} as an upgrade to the leather padding (Not Lazy Chainmail). Destructive leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +1 Entity Reach (Forge). Destructive leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 1 cracked stone bricks (Minecraft). In a leathersmithing table (Not Lazy Chainmail), destructive leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each destructive leather padding adds +1 Entity Reach (Forge), with a full set of armor with 12 destructive leather paddings giving +12 Entity Reach. Shields will only add Entity Reach when held in the main hand (Minecraft) or offhand (Minecraft).
