Discord Checkpoint

Discord Checkpoint is a viral marketing campaign by Discord (application), similar to Spotify Wrapped, starting on December 4, 2025. It allows users to see various statistic tracked by Discord. The following are the tracked data shown to the user:;;Total number of messages and a comparison with all other Discord users|Total number of emoji and top 5 emojis|Total hour in voice chat and a comparison with all other Discord users|Top 3 Discord server interacted with|Top 3 friend (Discord) interacted with and an option to direct message the closest friend|Top 5 video game played|Total number of Discord Quests completed;;Upon reading through the Checkpoint, an Avatar Decoration (Discord) will be given. A summary of the Checkpoint can be shared to others on the platform.
