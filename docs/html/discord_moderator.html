Discord moderator

A discord moderator, or discord mod, is a moderator for a Discord server on Discord (application). They often are volunteer, however, there are the occasional paid mods. Their main goal is to keep a server safe while keeping Discord user from breaking any Rule.&pThey are Stereotype as being obese, rude, strict, power-hungry, and overall antisocial. This includes silencing others in the name of mod disrespect and nitpicking over every violation.
