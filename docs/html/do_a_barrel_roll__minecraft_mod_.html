Do a Barrel Roll (Minecraft mod)

{{iDo a Barrel Roll}} is a Minecraft mod (Minecraft) made by enjarai which adds roll (flight), pitch (flight), and yaw when using an elytra to fly. This allows the player (Minecraft) to do many aerobatics, notably the barrel roll. The mod can be run client-side only, but it is recommended to also have it loaded server-side to allow other players to view the aerobatics.
