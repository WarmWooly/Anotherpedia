Do a Barrel Roll (Minecraft mod)

{{b{{iDo a Barrel Roll}}}} is a {{iMinecraft}} mod (Minecraft) made by enjarai which adds roll (flight), pitch (flight), and yaw when using an elytra to fly. This allows the player (Minecraft) to do many aerobatics, notably the barrel roll. The mod can be run client-side only, but it is recommended to also have it loaded server-side to allow other players to view the aerobatics. {{iDo a Barrel Roll}} is available for Minecraft: Java Edition 1.21.5, Minecraft: Java Edition 1.21.4, Minecraft: Java Edition 1.21.3, Minecraft: Java Edition 1.21.2, Minecraft: Java Edition 1.21.1, Minecraft: Java Edition 1.21, Minecraft: Java Edition 1.20.6, Minecraft: Java Edition 1.20.5, Minecraft: Java Edition 1.20.4, Minecraft: Java Edition 1.20.3, Minecraft: Java Edition 1.20.2, Minecraft: Java Edition 1.20.1, Minecraft: Java Edition 1.20, Minecraft: Java Edition 1.19.4, Minecraft: Java Edition 1.19.3, Minecraft: Java Edition 1.19.2, Minecraft: Java Edition 1.19.1, Minecraft: Java Edition 1.19, Minecraft: Java Edition 1.18.2, and Minecraft: Java Edition 1.17.1.
