{{iDo a Barrel Roll}} ({{iMinecraft}} mod)

<><><>{{b{{iDo a Barrel Roll}}}} is a [[{{iMinecraft}}]] [[mod|mod (Minecraft)]] made by [[enjarai]] which adds [[roll|roll (flight)]], [[pitch|pitch (flight)]], and [[yaw]] when using an [[elytra]] to fly. This allows the [[player|player (Minecraft)]] to do many [[aerobatics]], notably the [[barrel roll]].
