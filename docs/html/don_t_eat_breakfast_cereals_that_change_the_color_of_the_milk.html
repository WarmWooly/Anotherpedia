Don't eat breakfast cereals that change the color of the milk

"Don't eat breakfast cereals that change the color of the milk" is the 36th rule in the book Food Rules: An Eater's Manual by Michael Pollan. The saying is used to convey that it is better to avoid cereals that changes the color of milk added to it because of the chemical additives in the cereal.
