Draconic Leather Padding (Not Lazy Chainmail)

Draconic leather padding is an item in the Minecraft mod Not Lazy Chainmail as an upgrade to the toughened and inflated leather padding. Draconic leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving +2 Armor Toughness and -0.08 Gravity. Draconic leather padding can be melded in a Melder using 1 toughtened leather padding, 1 inflated leather padding, and 1 dragon head. In a leathersmithing table, draconic leather padding can be applied to chainmail armor or shields. Each draconic leather padding adds +2 Armor Toughness and -0.08 Gravity, with a full set of armor with 12 draconic leather paddings giving +24 Armor Toughness and -0.96 Gravity. Shields will only add Armor Toughness and remove Gravity when held in the main hand or offhand. If a set of armor has four or more draconic leather paddings applied, the advancement Off I Go, Apparently will be awarded.
