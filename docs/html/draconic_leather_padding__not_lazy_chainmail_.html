Draconic Leather Padding ({{iNot Lazy Chainmail}})

<><>||{{bStats}}||{{b[[Rarity|Rarity (Minecraft)]]}}|Uncommon||{{b[[Stack size|Stack size (Minecraft)]]}}|64||{{b[[Fire/lava&spresistance|Fire-resistant item (Minecraft)]]}}|Noinfo>>{{bDraconic leather padding}} is an [[item|item (Minecraft)]] in the {{i[[Minecraft]]}} [[mod|mod (Minecraft)]] {{i[[Not Lazy Chainmail]]}} as an upgrade to the [[toughened|toughened leather padding (Not Lazy Chainmail)]] and [[inflated leather padding|inflated leather padding (Not Lazy Chainmail)]]. Draconic leather padding is used to upgrade [[chainmail armor|chainmail armor (Not Lazy Chainmail)]] or [[shields|shield (Minecraft)]] in a [[leathersmithing table|leathersmithing table (Not Lazy Chainmail)]], giving +2 [[Armor Toughness|Armor Toughness (Minecraft)]] and -0.08 [[Gravity|Gravity (Forge)]].<><>Draconic leather padding can be melded in a [[Melder|Melder (Not Lazy Chainmail)]] using 1 [[toughtened leather padding|toughened leather padding (Not Lazy Chainmail)]], 1 [[inflated leather padding|inflated leather padding (Not Lazy Chainmail)]],and 1 [[dragon head|dragon head (Minecraft)]].<><>In a [[leathersmithing table|leathersmithing table (Not Lazy Chainmail)]], draconic leather padding can be applied to [[chainmail armor|chainmail armor (Not Lazy Chainmail)]] or [[shields|shield (Minecraft)]]. Each draconic leather padding adds +2 Armor Toughness and -0.08 Gravity, with a full set of armor with 12 draconic leather paddings giving +24 Armor Toughness and -0.96 Gravity. Shields will only add Armor Toughness and remove Gravity when held in the [[main hand|main hand (Minecraft)]] or [[offhand|offhand (Minecraft)]].
