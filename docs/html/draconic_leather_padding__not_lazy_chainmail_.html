Draconic Leather Padding (Not Lazy Chainmail)

<>Draconic leather padding is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the toughened leather padding (Not Lazy Chainmail) and inflated leather padding (Not Lazy Chainmail). Draconic leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +2 Armor Toughness (Minecraft) and -0.08 Gravity (Forge). Draconic leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 toughened leather padding (Not Lazy Chainmail), 1 inflated leather padding (Not Lazy Chainmail),and 1 dragon head (Minecraft). In a leathersmithing table (Not Lazy Chainmail), draconic leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each draconic leather padding adds +2 Armor Toughness and -0.08 Gravity, with a full set of armor with 12 draconic leather paddings giving +24 Armor Toughness and -0.96 Gravity. Shields will only add Armor Toughness and remove Gravity when held in the main hand (Minecraft) or offhand (Minecraft).
