Dynamic message sign

A dynamic message sign, also known as a variable message sign, abbreviation as DMS or VMS respectively, is an electronic sign used as a warning for traffic, routing information, safety information, and other important knowledge. A DMS has two phases, or Frame, that can be displayed at a time. The two phases will alternate, but not both phases are needed for a message to get across. Phases should only be used when needed. The time between each phase is determined by how long it takes to read a message and the speed of traffic, alongside sign visibility.

The message on each phase should be able to be understood separately. There is never a case to use three or more phases. There are also guidelines for the messages to be displayed on a DMS. The main goal is for drivers to be able to quickly make a decision with the new information. Abbreviations should be used sparingly to avoid confusion. Attempt to prevent two abbreviations in a row, and never have three in a row.
