Elara Voss

Elara Voss is a common AI tell when naming characters, especially in a fantasy setting, from AI chatbots. Many variations of the name exist, such as being denoted with Dr., having the first name Elena or Elias, having the last name Vox or Vance, or any combination of such. The name has been repeatedly used by LLMs when generating a name for a character in many contexts, which seems to be consistent across multiple public models. While the direct cause of such a phenomenon (rarely called the "Elara Effect") is not explicitly clear, its presence has been noted by both users of LLMs for generating creative writing and by others naturally noticing the trend of similar names appearing as authors, models, and fictional characters.
