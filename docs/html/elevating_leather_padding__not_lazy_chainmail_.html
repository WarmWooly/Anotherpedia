Elevating Leather Padding (Not Lazy Chainmail)

<>Elevating leather padding is an item in the Minecraft mod Not Lazy Chainmail as an upgrade to the leather padding. Elevating leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving +0.5 Step Height. Elevating leather padding can be melded in a Melder using 1 leather padding and 2 chains. In a leathersmithing table, elevating leather padding can be applied to chainmail armor or shields. Each elevating leather padding adds +0.5 Step Height, with a full set of armor with 12 elevating leather paddings giving +6 Step Height. Shields will only add Step Height when held in the main hand or offhand.
