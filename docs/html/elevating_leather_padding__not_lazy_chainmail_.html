Elevating Leather Padding (Not Lazy Chainmail)

<>{{bElevating leather padding}} is an item (Minecraft) in the {{iMinecraft}} mod (Minecraft) {{iNot Lazy Chainmail}} as an upgrade to the leather padding (Not Lazy Chainmail). Elevating leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +0.5 Step Height (Forge). Elevating leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 2 chain (Minecraft). In a leathersmithing table (Not Lazy Chainmail), elevating leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each elevating leather padding adds +0.5 Step Height (Forge), with a full set of armor with 12 elevating leather paddings giving +6 Step Height. Shields will only add Step Height when held in the main hand (Minecraft) or offhand (Minecraft).
