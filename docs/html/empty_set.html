For the symbol, see [[braces (symbol)]].}}&sp<>The {{bempty set}}, {{bnull set}}, or {{bvoid set}} is a [[finite|finite set]] [[set|set (mathematics)]] that contains no [[elements|element (mathematics)]]. The empty set has a [[cardinality]] (size) of 0 and can be represented with a pair of [[braces|braces (symbol)]], {{b<>}}, a [[zero]] with a slash, {{b&eset}}, or [[phi]].<><> The empty set itself is an element, so the set {&eset} is a set that contains the empty set. All sets will have the empty set as a [[subset]], and the only subset of the empty set is itself.
