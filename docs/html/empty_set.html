Empty set

"{}" redirects here. For the symbol, see braces (symbol).
The empty set, null set, or void set is a finite set that contains no elements. The empty set has a cardinality (size) of 0 and can be represented with a pair of braces, {}, a zero with a slash, &eset, or phi. The empty set itself is an element, so the set {&eset} is a set that contains the empty set. All sets will have the empty set as a subset, and the only subset of the empty set is itself.
