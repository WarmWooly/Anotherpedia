Enriched Alloy (Mekanism)

Enriched alloy is an item in the Minecraft mod Mekanism used to manufacture several blocks and items. It used to be called enriched alloy. Enriched alloy is created inside a Metallurgic Infuser using 1 redstone and 1 iron ingot.
