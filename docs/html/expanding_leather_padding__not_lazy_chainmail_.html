Expanding leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +0.33 Attack Knockback (Minecraft). Expanding leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 1 piston (Minecraft). Journeyman-level leatherworker (Minecraft) have a 50% chance to trade 1 leather padding (Not Lazy Chainmail) and 3 emerald (Minecraft) for 1 expanding leather padding. In a leathersmithing table (Not Lazy Chainmail), expanding leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft).
