ExplosiveSilverfish

{{bExplosiveSilverfish}} is a Minecraft Mod (Minecraft) made by Zero Criatividade that allows silverfish (Minecraft) to be fed gunpowder (Minecraft), causing them to explosion (Minecraft). ExplosiveSilverfish is available for Minecraft: Java Edition 1.20.1 and Minecraft: Java Edition 1.20 for Forge (Minecraft mod loader).
