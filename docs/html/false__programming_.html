False (programming)

<><>{{bFalse}} in [[programming]] refers to a [[logical statement]] where the [[outcome]] is [[incorrect]]. In [[binary]], it is represented as a 0. Most [[programming languages|programming language]] account for false as a [[data type]] named [[boolean]]. [[If statements|if statement]] that are found as false do not run the content of the statement.
