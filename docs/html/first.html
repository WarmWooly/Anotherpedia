First

For other uses of the term first, see first (disambiguation).
First is a phenomenon that precedes all other instances in a series. Anything first can't be before any other element. The direct item behind the first is known as second.

It is commonly seen as a success to be the first of something, such as the first to cross a finish line or the first to invent a new thing. Metals such as gold are used as a symbol for getting first place.
