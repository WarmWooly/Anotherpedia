Fixed storage (storage systems)

<>{{tFor [[data]] in [[computers|computer]], see [[fixed storage]].}}&sp<>{{bFixed storage}} is [[infrastructure]] used in [[storage systems|storage system]] that are designed to not be moved from where it is assembled. This is often done for [[organization]] and because it is much [[cheaper|cheap]] to keep all the [[storage]] in the same location.
