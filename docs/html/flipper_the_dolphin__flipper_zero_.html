Flipper the Dolphin (Flipper Zero)

Flipper the Dolphin is the mascot of the Flipper Zero and a virtual pet dolphin. Flipper the Dolphin is also called "Flipper", despite the dolphin not having an official name. Flipper is a curious cyborg and loves when the user uses the Flipper Zero on electronics. The user is able to rename Flipper to whatever they please. Flipper will gain XP for doing a variety of tasks. These tasks are based on the hardware on the system. Capturing and viewing a signal = 1 XP
Saving a signal = 3 XP
Recoring a RAW signal = 1 XP
Saving a RAW signal = 3 XP
Adding a signal manually = 2 XP
Emulating a saved signal = 2 XP
Using the Frequency Analyzer = 1 XP Entering the Read screen = 1 XP
Reading successfully = 3 XP
Adding an RFID manually = 2 XP
Saving an RFID tag = 3 XP
Emulating an RFID tag = 2 XP Entering the Read screen = 1
Using the Detect Reader = 1 XP
Reading successfully = 3 XP
Saving a card = 3 XP
Adding a card manually = 1 XP
Adding a key manually = 2 XP Reading successfully = 3 XP
Saving an IR signal = 3 XP
Emulating an IR signal = 1 XP Entering the USB-UART Bridge screen = 1 XP Entering the Read screen = 1 XP
Reading successfully = 3 XP
Saving an iButton key = 3 XP
Adding an iButton key manually = 2 XP
Emulating an iButton key = 2 XP Playing a Bad USB script = 3 XP Authorizing with the device = 3 XP Every Flipper starts at level 1. The highest level is level 3. Each level requires the following amount of XP:
Level 1 = 0 XP
Level 2 = 735 XP
Level 3 = 2940 XP

You can earn a total of 15 XP from each category per day for a total of 105 XP daily. This would require nearly a month of constant use to get to the max level. Because of the modability of the Flipper Zero, you can add more levels using external files. Happiness increases as you use the device, and decreases while you're not. There is no good easily accessible information about happiness, but you don't want Flipper sad, so you better scan your credit card until it's satisfied!
