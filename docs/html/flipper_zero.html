Flipper Zero

<><><>{{bFlipper Zero}} is an [[electronic]] [[device]] used to [[hack|Hacking]] into devices such as [[TVs|Television]], [[road signs|Dynamic Message Sign]], [[gates|Electronic Gate]], and [[ID scanners|ID Scanners]].<>The Flipper Zero has an orange/black [[LCD|Liquid-crystal Display]], a [[directional pad]], [[GPIO|General-purpose input/output]] pins, a [[MicroSD]] slot, a [[USB-C]] port, and an [[infrared transceiver]]. It also has a 433 [[MHz|Megahertz]] [[antenna|Antenna (electronics)]], a [[CC1101]] chip, an [[RFID|Radio-frequency Identification]], an [[NFC|Near Field Communication]], and [[Bluetooth]].&pThe 2000 [[mAh|Milliampere Hours]] [[battery]] can last roughly a [[week]] before needing to be [[recharged|Charging]]. There is also a [[mobile app|mobile application]] on the [[Apple App Store]] and on [[Google Play]].<>There are a lot of uses for the Flipper Zero. Usage may be considered [[illegal]] when used on property not owned by the person using the device.<>You have the ability to clone RFID and NFC using a Flipper Zero. These can be used to bypass many ID scanners that have little security.<>The infrared transceiver can replicate remotes for a wide array of appliances, including TVs, [[ACs|Air Conditioning Unit]], and [[all-in-one remotes|All-In-One Remote]].<>The GPIO pins allow use in programming and interacting with simple devices. This can also lead to adding attachments to make the Flipper Zero more powerful.<>A Flipper Zero can be made to brute-force [[passwords|Password]] to be able to log into devices unwillingly.<>{{tMain page: [[Flipper the Dolphin (Flipper Zero)]]}}&sp<>Inside the Flipper Zero is a [[virtual pet]] [[dolphin]], who also acts as the [[mascot]] to the Flipper Zero. Often called "Flipper" or "Flipper the Dolphin", despite not having an official name, is a [[cybernetic|Cyborg]] dolphin that loves when the Flipper Zero is used, and gets [[upset|Sadness]] when it's not. You can level it up by using the Flipper Zero for a variety of tasks.
