Flow Worm (Polarities)

The Flow Worm is an enemy that can be found in Limestone Caves in the Terraria mod Polarities. It acts like other worm enemies, burrowing before popping out to attack the player. On death, it drops 1 silver coin, a 50% for 1--3 Alkaline Fluid, and a 2% for Key Lime Pie.
