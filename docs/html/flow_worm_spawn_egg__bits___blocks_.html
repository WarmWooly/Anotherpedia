Flow Worm Spawn Egg (Bits & Blocks)

The flow worm spawn egg is a Creative item in the Minecraft mod Bits & Blocks used to instantly spawn flow worms. Using the flow worm spawn egg on a block will spawn a flow worm. When used in Survival, the flow worm spawn egg is consumed. A flow worm spawn egg fired from a dispenser will spawn a flow worm, using up the spawn egg. When used on a monster spawner, the spawner will then start spawning flow worms. The flow worm spawn egg can be renamed in an anvil, with spawned flow worms from the egg inheriting the name as if they were given a name tag.
