Friendslop

Friendslop is slang for a video game where the main appeal is drawn by playing with friends and the interactions generated though elements such as requiring teamwork and using proximity voice chat. Common examples of friendslop include Lethal Company, Content Warning, R.E.P.O., and Peak, which rely on players working together to achieve a goal while also giving a plethora of opportunities to mess with each other using the existing game mechanics. Friendslop is often used to criticize games that fall under its categorization, with proponents claiming that the gameplay of friendslop is lacking.
