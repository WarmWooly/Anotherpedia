Frozen Skewer (Bits & Blocks)

The frozen skewer is a throwable item in the Minecraft mod Bits & Blocks. The frozen skewer can be crafted with 8 skewers and 1 Ocean Mixtape (shapeless; makes 16). When used, the frozen skewer will be thrown in the direction the player is facing. If it hits a block, it will drop to the ground. If it hits a mob, it will deal 15 damage (7.5 hearts), apply Slowness II for 10 seconds, and be destroyed. Frozen skewers have an 8 tick cooldown between each throw.
