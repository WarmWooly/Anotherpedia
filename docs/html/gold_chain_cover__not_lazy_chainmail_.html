Gold Chain Cover (Not Lazy Chainmail)

<>The gold chain cover is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the leather padding (Not Lazy Chainmail). Gold chain covers are used to upgrade chainmail armor (Not Lazy Chainmail) in a leathersmithing table (Not Lazy Chainmail), making the armor type "Gold". Gold chain covers can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 1 block of gold chains (Not Lazy Chainmail). In a leathersmithing table (Not Lazy Chainmail), gold chain covers can be applied to chainmail armor (Not Lazy Chainmail). While up to 3 can be applied to a single item, the effects do not stack. Any curios (Not Lazy Chainmail) that scale based on worn gold chainmail armor (Not Lazy Chainmail) will also scale for armor with a gold chain cover.
