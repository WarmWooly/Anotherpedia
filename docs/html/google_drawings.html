Google Drawings

Often confused with Google Doodle.&spGoogle Drawings is a diagramming software developed by Google which can both be accessed by the browser and as a standalone application on ChomeOS, which is a part of Google Docs Editors. It was released on April 12, 2010, as Google Docs Drawings. The following are all actions that are listed within Google Drawings <>. Under the file tab, there are several actions relating to files and file management.&spNew - Creates a new drawing.&spOpen (Ctrl + O) - The user can select a drawing from their Google Drive to open.&spMake a copy - Create a copy of the file, where the user can share it with the same people and keep/remove comments.&spShare - The user to share the drawing with other people, either using their specific accounts or as a link.&spEmail - The user can send the drawing in an email.&spDownload - The user cam download the drawing as a PDF, PNG, SVG, or a JPG.&spRename - Renames the drawing.&spMove to trash - Moves the drawing into the user's trash.&spVersion history - The user can see any edits made to the drawing.Make available offline - The user can use and edit the file while they are offline.&spDetails - Displays document details, such as the location, owner, creation date, and last edit date.&spLanguage - Changes the language of Google Drawings.&spPage setup - Changes the length and width of the drawing, with presets of Standard 4:3, Widescreen 16:9, and Widescreen 16:10.&spPrint preview - Shows the drawing as if it was to be printed.&spPrint (Ctrl + P) - Prints the drawing. Under the edit tab, there are several actions relating to general file editing.&spUndo (Ctrl + Z) - Reverses the last action.&spRedo (Ctrl + Y) - Reverses an undo, performing the last action undone.&spCut (Ctrl + X) - Removes the selected elements, which can then be pasted.&spCopy (Ctrl + C) - Saves the selected elements, which can then be pasted.&spPaste (Ctrl + V) - Makes a clone of the saved elements, putting it onto the drawing.&spPaste without formatting (Ctrl + Shift + V) - Makes a clone of the saved elements, putting it onto the drawing, where all text formatting is set to the formatting determined in the element.&spSelect all (Ctrl + A) - Selects all elements in the drawing.&spDelete (Backspace) - Removes the selected elements.&spDuplicate (Ctrl + D) - Makes a clone of the selected elements, putting it onto the drawing.&spFind and replace (Ctrl + H) - Finds all instances of a specified string of text in the drawing, which can be replaced with a different string of text. Under the view tab, there are several actions related to tool visibility.&spShow ruler - Shows/hides the ruler along the edges of the drawing.&spGuides:&sp&tabShow guides (Ctrl + Alt + E, Ctrl + Alt + G) - Shows all guides.&sp&tabAdd vertical guide (Ctrl + Alt + E, Ctrl + Alt + V) - Creates a new vertical guide.&sp&tabAdd horizontal guide (Ctrl + Alt + E, Ctrl + Alt + H) - Creates a new horizontal guide.&sp&tabEdit guides (Ctrl + Alt + E, Ctrl + Alt + W) - Allows the creation, deletion, positioning, and coloring of all guides.&sp&tabClear guides - Removes all guides.&spSnap to - Sets element snapping to a grid or guides.&spShow comments (Ctrl + Alt + Shift + J) - Shows/hides comments.&spZoom menu - Gives the ability to set the zoom, such as to fit the screen (Ctrl + Alt + [ ), to the original size (Ctrl + O), to zoom in (Ctrl + =) and to zoom out (Ctrl + -).&spFull screen - Puts the drawing into full-screen mode. Under the insert tab, there are several actions related to element creation.&spImage - Creates an image from the device, Google images, Google Drive, Google Photos, the device's camera or a URL.&spText box - Creates a new text box.&spShape - Creates an editable shape, arrow. callout, or equation symbol.&spTable - Creates a new table of a specified number of columns and rows.&spChart - Creates a chart, such as a bar graph, column chart, line graph, or pie chart. The user can also use a chart made in Google Sheets.&spDiagram - Inserts a diagram, such as a grid, hierarchy, timeline, process, relationship, or cycle diagram.&spWord art - Turns text into an editable shape.&spLine - Creates lines, such as a normal line, an arrow, an elbow and curved connector, curved lines, shapes with lines, and a free draw.&spSpecial characters - Inputs special characters (such as emojis) into text.&spLink (Ctrl + K) - Turns a URL into a link.&spComment (Ctrl + Alt + M) - Creates a new comment. Under the format tab, there are several actions related to element formatting.&spText:&sp&tabBold (Ctrl + B) - Used to create bold text.&sp&tabItalic (Ctrl + I) - Used to create italic text.&sp&tabUnderline (Ctrl + U) - Used to create uunderlined text.&sp&tabStrikethrough (Alt + Shift + 5) - Used to create sstriked text.&sp&tabSuperscript (Ctrl + .) - Used to create s-psupercripts.&sp&tabSubscript (Ctrl + ,) - Used to create s-bsubscripts.&sp&tabSize - Changes the size of the text, which can be increased with (Ctrl + Shift + .), and decreased with (Ctrl + Shift + ,).&sp&tabColor - Changes the color of the text.&sp&tabHighlight color - Changes the color of the highlight of the text.&sp&tabCapitalization - Changes the capitalization to all lowercase, all uppercase, or title case.&spAlign & indent:&sp&tabLeft (Ctrl + Shift + L) - Horizontally aligns the text to the left of the element.&sp&tabCenter (Ctrl + Shift + E) - Horizontally aligns the text to the center of the element.&sp&tabRight (Ctrl + Shift + R) - Horizontally aligns the text to the right of the element.&sp&tabJustified (Ctrl + Shift + J) - Horizontally aligns the text to the left of the element, where the text is spaced so the right edge lines up with the right bound of the element.&sp&tabIncrease indent (Ctrl + ]) - Increases the indent by 1 tab, or 0.5 inches.&sp&tabDecrease indent (Ctrl + [) - Decreases the indent by 1 tab, or 0.5 inches.&sp&tabTop - Vertically aligns the text to the top of the element.&sp&tabMiddle - Vertically aligns the text to the center of the element.&sp&tabBottom - Vertically aligns the text to the bottom of the element.&spLine & paragraph spacing - Changes the spacing between line and before/after paragraphs.&spBullets & numbering - Used to create bulleted and numbered lists, alongside editing the list's properties.&spTable - Allows the user to add/remove columns and rows, distribute columns and rows, and merge/unmerge cells.&spImage - Crops images, replaces images, or resets formatting on images.&spBorders & lines - Changes the color, thickness, style, and decorations on lines and borders, alongside changing the size of an arrowhead.&spFill color - Changes the inside color of a shape into a different color or gradient.&spChange connector - Changes a connector into a straight line, an elbow connector, or a curved connector.&spReroute connector - Gives the ability to change the start and end connections on a connector.&spFormat options - Opens the formatting menu, giving access to change the size and rotation, position, drop shadow, reflection, and alt text.&spClear formatting (Ctrl + ) - Removes all text formatting on text. Under the arrange tab, there are several actions related to element positioning.&spOrder:&sp&tabMove to front (Ctrl + Shift + Up Arrow) - Moves the selected element to the top layer.&sp&tabMove forward (Ctrl + Up Arrow) - Moves the selected element up a layer.&sp&tabSend backward (Ctrl + Down Arrow) - Moves the selected element down a layer.&sp&tabSend to back (Ctrl + Shift + Down Arrow) - Moves the selected element to the bottom layer.&spAlign - Used to align multiple elements horizontally and vertically to each other.&spDistribute - Used to align multiple elements based on a distribution between each other.&spCenter on page - Used to center an element horizontally or vertically in the drawing.&spRotate - Used to flip or rotate an element.&spGroup (Ctrl + Alt + G) - Used to combine multiple elements into a single element.&spUngroup (Ctrl + Alt + Shift + G) - Used to turn a group back into multiple elements. Under the tools tab, there are several actions related to miscellaneous tools.&spSpelling:&sp&tabSpell check (Ctrl + Alt + X) - Checks and offers suggestions for spelling and grammar.&sp&tabShow spelling suggestions - Shows/hides spelling and grammar suggestions.&sp&tabPersonal dictionary - Shows the user's words to not be marked down as spelled incorrectly by the spell check.&spExplore (Ctrl + Shift + I) - Opens Explore.&spLinked objects - Shows files linked to the drawing.&spDictionary (Ctrl + Shift + Y) - Opens a dictionary to find definitions of words.&spNotification settings - Changes the criteria for comments to send a notification.&spPreferences - Changes preferences, such as automatic capitalization, markdown detection, and text substitution rules.&spAccessibility - Accesses the accessibility settings for the visually impaired. Under the help tab, there are several actions related to help related to Google Drawings.&spSearch the menus (Alt + /) - Searches through all the tab menus based on the input query.&spHelp - Opens a help menu to aid users who don't know how to do a specific task or action.&spReport a problem - Creates a report about a bug or issue about Google Drawings.&spReport abuse - Creates a report about a violation of Google's Terms of Service.&spPrivacy Policy - Opens Google's Privacy Policy.&spTerms of Service - Opens Google's Terms of Service.&spKeyboard shortcuts (Ctrl + /) - Lists all keyboard shortcuts that can be performed.
