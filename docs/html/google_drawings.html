Google Drawings

Often confused with Google Doodle.
Google Drawings is a diagramming software developed by Google which can both be accessed by the browser and as a standalone application on ChomeOS, which is a part of Google Docs Editors. It was released on April 12, 2010, as Google Docs Drawings. The following are all actions that are listed within Google Drawings <>. Under the file tab, there are several actions relating to files and file management.
New - Creates a new drawing.
Open (Ctrl + O) - The user can select a drawing from their Google Drive to open.
Make a copy - Create a copy of the file, where the user can share it with the same people and keep/remove comments.
Share - The user to share the drawing with other people, either using their specific accounts or as a link.
Email - The user can send the drawing in an email.
Download - The user cam download the drawing as a PDF, PNG, SVG, or a JPG.
Rename - Renames the drawing.
Move to trash - Moves the drawing into the user's trash.
Version history - The user can see any edits made to the drawing.Make available offline - The user can use and edit the file while they are offline.
Details - Displays document details, such as the location, owner, creation date, and last edit date.
Language - Changes the language of Google Drawings.
Page setup - Changes the length and width of the drawing, with presets of Standard 4:3, Widescreen 16:9, and Widescreen 16:10.
Print preview - Shows the drawing as if it was to be printed.
Print (Ctrl + P) - Prints the drawing. Under the edit tab, there are several actions relating to general file editing.
Undo (Ctrl + Z) - Reverses the last action.
Redo (Ctrl + Y) - Reverses an undo, performing the last action undone.
Cut (Ctrl + X) - Removes the selected elements, which can then be pasted.
Copy (Ctrl + C) - Saves the selected elements, which can then be pasted.
Paste (Ctrl + V) - Makes a clone of the saved elements, putting it onto the drawing.
Paste without formatting (Ctrl + Shift + V) - Makes a clone of the saved elements, putting it onto the drawing, where all text formatting is set to the formatting determined in the element.
Select all (Ctrl + A) - Selects all elements in the drawing.
Delete (Backspace) - Removes the selected elements.
Duplicate (Ctrl + D) - Makes a clone of the selected elements, putting it onto the drawing.
Find and replace (Ctrl + H) - Finds all instances of a specified string of text in the drawing, which can be replaced with a different string of text. Under the view tab, there are several actions related to tool visibility.
Show ruler - Shows/hides the ruler along the edges of the drawing.
Guides:
&tabShow guides (Ctrl + Alt + E, Ctrl + Alt + G) - Shows all guides.
&tabAdd vertical guide (Ctrl + Alt + E, Ctrl + Alt + V) - Creates a new vertical guide.
&tabAdd horizontal guide (Ctrl + Alt + E, Ctrl + Alt + H) - Creates a new horizontal guide.
&tabEdit guides (Ctrl + Alt + E, Ctrl + Alt + W) - Allows the creation, deletion, positioning, and coloring of all guides.
&tabClear guides - Removes all guides.
Snap to - Sets element snapping to a grid or guides.
Show comments (Ctrl + Alt + Shift + J) - Shows/hides comments.
Zoom menu - Gives the ability to set the zoom, such as to fit the screen (Ctrl + Alt + [ ), to the original size (Ctrl + O), to zoom in (Ctrl + =) and to zoom out (Ctrl + -).
Full screen - Puts the drawing into full-screen mode. Under the insert tab, there are several actions related to element creation.
Image - Creates an image from the device, Google images, Google Drive, Google Photos, the device's camera or a URL.
Text box - Creates a new text box.
Shape - Creates an editable shape, arrow. callout, or equation symbol.
Table - Creates a new table of a specified number of columns and rows.
Chart - Creates a chart, such as a bar graph, column chart, line graph, or pie chart. The user can also use a chart made in Google Sheets.
Diagram - Inserts a diagram, such as a grid, hierarchy, timeline, process, relationship, or cycle diagram.
Word art - Turns text into an editable shape.
Line - Creates lines, such as a normal line, an arrow, an elbow and curved connector, curved lines, shapes with lines, and a free draw.
Special characters - Inputs special characters (such as emojis) into text.
Link (Ctrl + K) - Turns a URL into a link.
Comment (Ctrl + Alt + M) - Creates a new comment. Under the format tab, there are several actions related to element formatting.
Text:
&tabBold (Ctrl + B) - Used to create bold text.
&tabItalic (Ctrl + I) - Used to create italic text.
&tabUnderline (Ctrl + U) - Used to create uunderlined text.
&tabStrikethrough (Alt + Shift + 5) - Used to create sstriked text.
&tabSuperscript (Ctrl + .) - Used to create s-psupercripts.
&tabSubscript (Ctrl + ,) - Used to create s-bsubscripts.
&tabSize - Changes the size of the text, which can be increased with (Ctrl + Shift + .), and decreased with (Ctrl + Shift + ,).
&tabColor - Changes the color of the text.
&tabHighlight color - Changes the color of the highlight of the text.
&tabCapitalization - Changes the capitalization to all lowercase, all uppercase, or title case.
Align & indent:
&tabLeft (Ctrl + Shift + L) - Horizontally aligns the text to the left of the element.
&tabCenter (Ctrl + Shift + E) - Horizontally aligns the text to the center of the element.
&tabRight (Ctrl + Shift + R) - Horizontally aligns the text to the right of the element.
&tabJustified (Ctrl + Shift + J) - Horizontally aligns the text to the left of the element, where the text is spaced so the right edge lines up with the right bound of the element.
&tabIncrease indent (Ctrl + ]) - Increases the indent by 1 tab, or 0.5 inches.
&tabDecrease indent (Ctrl + [) - Decreases the indent by 1 tab, or 0.5 inches.
&tabTop - Vertically aligns the text to the top of the element.
&tabMiddle - Vertically aligns the text to the center of the element.
&tabBottom - Vertically aligns the text to the bottom of the element.
Line & paragraph spacing - Changes the spacing between line and before/after paragraphs.
Bullets & numbering - Used to create bulleted and numbered lists, alongside editing the list's properties.
Table - Allows the user to add/remove columns and rows, distribute columns and rows, and merge/unmerge cells.
Image - Crops images, replaces images, or resets formatting on images.
Borders & lines - Changes the color, thickness, style, and decorations on lines and borders, alongside changing the size of an arrowhead.
Fill color - Changes the inside color of a shape into a different color or gradient.
Change connector - Changes a connector into a straight line, an elbow connector, or a curved connector.
Reroute connector - Gives the ability to change the start and end connections on a connector.
Format options - Opens the formatting menu, giving access to change the size and rotation, position, drop shadow, reflection, and alt text.
Clear formatting (Ctrl + ) - Removes all text formatting on text. Under the arrange tab, there are several actions related to element positioning.
Order:
&tabMove to front (Ctrl + Shift + Up Arrow) - Moves the selected element to the top layer.
&tabMove forward (Ctrl + Up Arrow) - Moves the selected element up a layer.
&tabSend backward (Ctrl + Down Arrow) - Moves the selected element down a layer.
&tabSend to back (Ctrl + Shift + Down Arrow) - Moves the selected element to the bottom layer.
Align - Used to align multiple elements horizontally and vertically to each other.
Distribute - Used to align multiple elements based on a distribution between each other.
Center on page - Used to center an element horizontally or vertically in the drawing.
Rotate - Used to flip or rotate an element.
Group (Ctrl + Alt + G) - Used to combine multiple elements into a single element.
Ungroup (Ctrl + Alt + Shift + G) - Used to turn a group back into multiple elements. Under the tools tab, there are several actions related to miscellaneous tools.
Spelling:
&tabSpell check (Ctrl + Alt + X) - Checks and offers suggestions for spelling and grammar.
&tabShow spelling suggestions - Shows/hides spelling and grammar suggestions.
&tabPersonal dictionary - Shows the user's words to not be marked down as spelled incorrectly by the spell check.
Explore (Ctrl + Shift + I) - Opens Explore.
Linked objects - Shows files linked to the drawing.
Dictionary (Ctrl + Shift + Y) - Opens a dictionary to find definitions of words.
Notification settings - Changes the criteria for comments to send a notification.
Preferences - Changes preferences, such as automatic capitalization, markdown detection, and text substitution rules.
Accessibility - Accesses the accessibility settings for the visually impaired. Under the help tab, there are several actions related to help related to Google Drawings.
Search the menus (Alt + /) - Searches through all the tab menus based on the input query.
Help - Opens a help menu to aid users who don't know how to do a specific task or action.
Report a problem - Creates a report about a bug or issue about Google Drawings.
Report abuse - Creates a report about a violation of Google's Terms of Service.
Privacy Policy - Opens Google's Privacy Policy.
Terms of Service - Opens Google's Terms of Service.
Keyboard shortcuts (Ctrl + /) - Lists all keyboard shortcuts that can be performed.
