Google Search performance insights

<>
Google Search performance insights are a panel that can appear in the search engine Google Search that appears on search terms where a user's website appears based on the given query. It lists the website being tracked and gives a dropdown to view its performance based on the last 7, 28, and 90 days. Below, it shows the clicks, impressions, and average position, with indicators for an increase or decrease in appearance in search results. The bottom bar takes the user to the Insights tab on Google Search Console for their site. Its kebab menu (three dots) allows the user to hide the insight for that specific website, hide for all websites, or send feedback.
