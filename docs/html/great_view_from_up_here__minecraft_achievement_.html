Great View From Up Here (Minecraft achievement)

For the achievement in aiMinecraft: New Nintendo 3DS Edition or aiLegacy Console editions, see Great View From Up Here (aiMinecraft: Legacy Console & aiNew Nintendo 3DS achievement), and for the advancement in Minecraft: Java Edition, see Great View From Up Here (Minecraft advancement).
Great View From Up Here is an achievement available on Minecraft: Bedrock Edition that is earned when the player levitates 50 or more blocks. The description is as follows:Levitate up 50 blocks from the attacks of a ShulkerUpon completing the achievement, the player will receive 20 G (on Xbox) or a bronze trophy (on PlayStation), alongside the "Elder Dragon Wings" back item for the player's skin.
