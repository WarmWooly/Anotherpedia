Grenade Launcher (Laser Fight)

<>The Grenade Launcher is a grenade-firing gun in Laser Fight. The grenades cause a large explosion, killing those within their radius while disorienting those nearby. The grenades arc, allowing the user to shoot over walls to hit the target. Killing someone with the Grenade Launcher will award the badge Souper Bobm Survivl.
