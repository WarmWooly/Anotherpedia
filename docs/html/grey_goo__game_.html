Grey Goo (game)

{{tThis is often confused for the fictional nanobots of a similar name, gray goo.}}&sp{{b{{iGrey Goo}}}} is a real-time strategy game video game released in 2015 developed by Petroglyph Games and published by Grey Box. The game is set in a science fiction universe made up of multiples faction. The main gameplay is focused around the four playable factions: the Humans, the Beta, the Goo, and the Shroud. This game supported both single-player game and multiplayer game. Each faction had different buildings, units, and play styles based on the faction, alongside unique segments to the campaign.
