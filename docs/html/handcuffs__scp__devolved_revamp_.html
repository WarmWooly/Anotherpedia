Handcuffs (SCP: Devolved Revamp)

<>The Handcuffs are a primary tool within SCP: Devolved Revamp for the Administrative Branch (SCP: Devolved Revamp) used to arrest Map (SCP: Devolved Revamp) player (roblox) who break the rules. Arresting a player on any other Role (SCP: Devolved Revamp) will put them on Class D (SCP: Devolved Revamp) for 3 minutes. The cuffs will have a 60-second cooldown after use before they can be used again.Reasons for Foundation Staff Arrests:
- Breaching: intentionally letting out SCPs
- Conspiracy: helping Class D escape or cross-teaming with the Chaos Insurgency
- Negligence: refusing orders to perform expected and reasonable, role-specific duties
