Healthy Leather Padding (Not Lazy Chainmail)

<>Healthy leather padding is an item in the Minecraft mod Not Lazy Chainmail as an upgrade to the leather padding. Healthy leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving +2 Max Health (1 heart). Healthy leather padding can be melded in a Melder using 1 leather padding and 1 rose gold ingot. In a leathersmithing table, healthy leather padding can be applied to chainmail armor or shields. Each healthy leather padding adds +2 Max Health, with a full set of armor with 12 healthy leather paddings giving +24 Max Health (12 hearts). Shields will only add Max Health when held in the main hand or offhand.
