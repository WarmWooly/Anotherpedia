Healthy Leather Padding (Not Lazy Chainmail)

<>Healthy leather padding is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the leather padding (Not Lazy Chainmail). Healthy leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +2 Max Health (Minecraft) (1 heart). Healthy leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 1 rose gold ingot (Not Lazy Chainmail). In a leathersmithing table (Not Lazy Chainmail), healthy leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each healthy leather padding adds +2 Max Health, with a full set of armor with 12 healthy leather paddings giving +24 Max Health (12 hearts). Shields will only add Max Health when held in the main hand (Minecraft) or offhand (Minecraft).
