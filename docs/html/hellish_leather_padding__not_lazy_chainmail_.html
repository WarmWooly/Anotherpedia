Hellish Leather Padding (Not Lazy Chainmail)

<>Hellish leather padding is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the leather padding (Not Lazy Chainmail). Hellish leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +1 Hellishness and -0.1 Swim Speed (Forge). With Hellishness, upon entering fire (Minecraft), lava (Minecraft), or a campfire (Minecraft), Fire Resistance (Minecraft) will be applied for 2 * Hellishness seconds. Initial and lingering damage do not get prevented. Hellish leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 2 hellish glass (Not Lazy Chainmail). In a leathersmithing table (Not Lazy Chainmail), hellish leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each hellish leather padding adds +1 Hellishness and -0.1 Swim Speed, with a full set of armor with 12 hellish leather paddings giving +12 Hellishness and -1.2 Swim Speed. Shields will only add Hellishness and remove Swim Speed when held in the main hand (Minecraft) or offhand (Minecraft).
