Hollow Dirt (Bits & Blocks)

Hollow dirt is a block in the Minecraft mod Bits & Blocks used to spawn flow worms. Hollow dirt can be crafted with 1 Flow Worms and 1 dirt/coarse dirt/rooted dirt (shapeless). When placed, a hollow dirt will spawn a flow worm. Only one flow worm can spawn per hollow dirt. On breaking, a block of dirt will be dropped instead.
