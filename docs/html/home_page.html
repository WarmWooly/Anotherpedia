Home page

<>{{tFor the home page of [[Anotherpedia]], see [[main page]].}}&sp<>A {{bhome page}}, also written {{bhomepage}}, is the main [[web page]] on a [[website]], which is often at the root of the [[domain|domain name]]. It is common to use the home page to also be the [[landing page]] on a website, drawing users in as soon as they enter. [[Designing|web design]] is important, such as keeping up-to-date information about the website in a place that is easy to locate.
