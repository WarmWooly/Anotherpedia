How to make/edit pages

Want to make a page, or add something new to one that currently exists? Anyone can do it (yes, anyone, on {{ianything}}).&pMaking a page is easy, only requiring some time and effort. Just like the non-random counterpart Wikipedia, Anotherpedia has different tags that can be used to do a variety of things, such as adding links, images, and styling. To create/edit a page, click "Edit Page" or type in the search bar for a page that does not exist yet. Main page: Link (Anotherpedia)&spLinks are vital for finding information. Links are created by surrounding text in a pair of square brackets, also known as []. Surrounding text in brackets creates a link:&spPerson = Person&spIn cases where you want to have the text of the link differ from where the link goes to, add a bar in the middle.&spPeople = People&spAttempt to only use a link once per page on the first instance of the term, unless required due to page length. To add paragraph breaks, use an ampersand, displayed as &, behind a p or an sp: &p, &sp. They each represent a full break (2 line breaks) codeAlt + P and a half break (1 line break) codeAlt + S respectively.
