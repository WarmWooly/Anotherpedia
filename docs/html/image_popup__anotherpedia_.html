Image popup (Anotherpedia)

{{bImage popups}} are separate window (browser) that are opened when an image is right click on a web browser in Anotherpedia. This creates a popup of the image which can then be manipulated using these keybinding:&p{{bRight click}} - Set the size of the window to the image.&sp{{bZ/A}} - Flip the image horizontal.&sp{{bShift + Z}} - Flip the image slightly horizontally.&sp{{bShift + A}} - Stretch the image horizontally.&sp{{bX/S}} - Flip the image vertical.&sp{{bShift + X}} - Flip the image slightly vertically.&sp{{bShift + S}} - Stretch the image vertically.&sp{{bR}} - Rotate the image clockwise.&sp{{bShift + R}} - Rotate the image slightly clockwise.&sp{{bF}} - Rotate the image counterclockwise.&sp{{bShift + F}} - Rotate the image slightly counterclockwise.&sp{{bQ}} - Reset the image.
