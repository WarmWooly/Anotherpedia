Image popup (Anotherpedia)

Image popups are separate windows that are opened when an image is right-clicked on a browser in Anotherpedia. This creates a popup of the image which can then be manipulated using these keybindings:&pRight click - Set the size of the window to the image.&spZ/A - Flip the image horizontally.&spShift + Z - Flip the image slightly horizontally.&spShift + A - Stretch the image horizontally.&spX/S - Flip the image vertically.&spShift + X - Flip the image slightly vertically.&spShift + S - Stretch the image vertically.&spR - Rotate the image clockwise.&spShift + R - Rotate the image slightly clockwise.&spF - Rotate the image counterclockwise.&spShift + F - Rotate the image slightly counterclockwise.&spQ - Reset the image.
