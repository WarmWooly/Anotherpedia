Image popup (Anotherpedia)

Image popups are separate windows that are opened when an image is right-clicked on a browser in Anotherpedia. This creates a popup of the image which can then be manipulated using these keybindings:

Right click - Set the size of the window to the image.
Z/A - Flip the image horizontally.
Shift + Z - Flip the image slightly horizontally.
Shift + A - Stretch the image horizontally.
X/S - Flip the image vertically.
Shift + X - Flip the image slightly vertically.
Shift + S - Stretch the image vertically.
R - Rotate the image clockwise.
Shift + R - Rotate the image slightly clockwise.
F - Rotate the image counterclockwise.
Shift + F - Rotate the image slightly counterclockwise.
Q - Reset the image.
