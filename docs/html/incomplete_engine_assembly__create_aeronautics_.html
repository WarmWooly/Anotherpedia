Incomplete Engine Assembly (Create Aeronautics)

The incomplete engine assembly is an item in the Minecraft mod Create Aeronautics that is used to represent an engine assembly mid-processing. It can be created when an iron sheet is first put into a mechanical saw, and it is turned into an engine assembly (or byproduct) immediately after being pressed for the 8th time.
