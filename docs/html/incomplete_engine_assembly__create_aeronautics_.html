Incomplete Engine Assembly (Create Aeronautics)

<>The incomplete engine assembly is an item (Minecraft) in the Minecraft mod (Minecraft) Create Aeronautics that is used to represent an engine assembly (Create Aeronautics) mid-processing. It can be created when an iron sheet (Create) is first put into a mechanical saw (Create), and it is turned into an engine assembly (or byproduct) immediately after being mechanical press (Create) for the 8th time.
