Incomplete Gyroscopic Mechanism (Create Aeronautics)

The incomplete gyroscopic mechanism is an item in the Minecraft mod Create Aeronautics that is used to represent a gyroscopic mechanism mid-processing. It can be created when an iron sheet first has a cogwheel deployed on it, and it is turned into a gyroscopic mechanism (or byproduct) immediately after having a brass nugget deployed on it for the 5th time.
