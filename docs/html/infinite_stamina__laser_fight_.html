Infinite Stamina (Laser Fight)

{{bInfinite Stamina}} is a mode (Laser Fight) in {{iLaser Fight (Roblox game)}} where the refill rate for stamina (Laser Fight) is almost instant and always running. This means that advanced movement (Laser Fight) can be performed regardless of the player's stamina management. Players will spawn in with the Basic Gun (Laser Fight).
