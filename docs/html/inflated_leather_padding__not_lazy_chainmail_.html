Inflated Leather Padding (Not Lazy Chainmail)

<>Inflated leather padding is an item in the Minecraft mod Not Lazy Chainmail as an upgrade to the leather padding. Inflated leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving -0.01 Gravity. Inflated leather padding can be melded in a Melder using 1 leather padding and 1 phantom membrane. In a leathersmithing table, inflated leather padding can be applied to chainmail armor or shields. Each inflated leather padding removes -0.01 Gravity, with a full set of armor with 12 inflated leather paddings giving -0.12 Gravity. Shields will only remove Gravity when held in the main hand or offhand.
