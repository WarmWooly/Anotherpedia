Inflated Leather Padding (Not Lazy Chainmail)

<>Inflated leather padding is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the leather padding (Not Lazy Chainmail). Inflated leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving -0.01 Gravity (Forge). Inflated leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 1 phantom membrane (Minecraft). In a leathersmithing table (Not Lazy Chainmail), inflated leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each inflated leather padding removes -0.01 Gravity, with a full set of armor with 12 inflated leather paddings giving -0.12 Gravity. Shields will only remove Gravity when held in the main hand (Minecraft) or offhand (Minecraft).
