Integer

An integer is a number that is natural number (1, 2, 3...), 0 (0), or negative number of a natural number (-1, -2, -3...). The set (mathmatics) of all integers is denoted with bold (typography) Z or blackboard bold &bbZ. The set of integers contains the subset of natural numbers &bbN and is itself a subset of rational number &bbQ
