Invincible Bees

Invincible Bees is a Minecraft resource pack based on the "We can be bees" meme from Invincible, uploaded by SirJumpy8772 on June 1, 2025. The pack changes the texture of bees to be Invincible (with two variations), Omni-Man, or Rex Splode. It requires OptiFine or Entity Texture Features and Entity Model Features to render correctly. It is available on CurseForge for all major versions from 1.20 to 26.1.
