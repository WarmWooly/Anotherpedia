Iron Composite Chainmail Belt (Not Lazy Chainmail)

<>The iron composite chainmail belt is a belt curios (Not Lazy Chainmail) in the Minecraft mod (Minecraft) Not Lazy Chainmail that allows the wearer to teleport to 3 set locations instantly. The iron composite chainmail belt can be created through chainmail project (Not Lazy Chainmail) with the settings "Iron Composigte Belt". The belt takes 128 iron composite chain link (Not Lazy Chainmail) to craft. When the iron composite chainmail belt is worn in a belt slot, the user can press a keybind (Minecraft) (codeK by default) to open up the belt's teleport menu. The user can save a location in the Overworld (Minecraft), Nether (Minecraft), or End (Minecraft) and teleport to those locations using the "Save" and "Teleport" buttons, respectively. Teleporting makes a fwoosh sound at the start and end points of the teleport. By default, teleports are set to 0, 100, 0 XYZ in the Overworld. Other modded dimensions will not work, defaulting to the Overworld. Iron composite chainmail belts can be Crusher (Not Lazy Chainmail) into unfinished iron composite chainmail project (Not Lazy Chainmail) containing 128 iron composite chain link (Not Lazy Chainmail).
