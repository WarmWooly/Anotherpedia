Iron Composite Sieve Mesh (Not Lazy Chainmail)

The iron composite sieve mesh is an item in the Minecraft mod Not Lazy Chainmail used for sieving blocks into various resources. The iron composite sieve mesh is the highest tier, having unique, rare drops for valuable items. The iron composite sieve mesh can be crafted with 1 iron composite chain carpet and 8 obsidian (place the carpet in the center and surround with obsidian). For the sieving loot table and applicable blocks, see sieving ({{aiNot Lazy Chainmail).}}
Hold the iron composite sieve mesh in the offhand with the block to be sieved in the main hand. Upon using the iron composite sieve mesh, the block will be consumed, and any items collected will be placed in the inventory or on the ground if there is not enough space. The Sieve Dropper is a block that can be used to automatically sieve items that pass above on a chain conveyor. By inputting the iron composite sieve mesh, the valid blocks will automatically be sieved and dropped below. The iron composite sieve mesh will still lose durability while in a sieve dropper.
