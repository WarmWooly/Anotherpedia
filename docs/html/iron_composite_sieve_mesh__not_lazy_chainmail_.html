Iron Composite Sieve Mesh (Not Lazy Chainmail)

<>The iron composite sieve mesh is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail used for sieving (Not Lazy Chainmail) block (Minecraft) into various resources. The iron composite sieve mesh is the highest tier, having unique, rare drops for valuable items. The iron composite sieve mesh can be crafting (Minecraft) with 1 iron composite chain carpet (Not Lazy Chainmail) and 8 obsidian (Minecraft) (place the carpet in the center and surround with obsidian). For the sieving loot table and applicable blocks, see sieving ({{aiNot Lazy Chainmail).}}
Hold the iron composite sieve mesh in the offhand (Minecraft) with the block to be sieved in the main hand (Minecraft). Upon using the iron composite sieve mesh, the block will be consumed, and any items collected will be placed in the inventory (Minecraft) or on the ground if there is not enough space. The Sieve Dropper (Not Lazy Chainmail) is a block (Minecraft) that can be used to automatically sieve items that pass above on a chain conveyor (Not Lazy Chainmail). By inputting the iron composite sieve mesh, the valid blocks will automatically be sieved and dropped below. The iron composite sieve mesh will still lose durability while in a sieve dropper. An iron composite sieve mesh can be enchanting (Minecraft) with Unbreaking (Minecraft), Mending (Minecraft), and Curse of Vanishing (Minecraft), using an enchanted book (Minecraft) on an anvil (Minecraft).
