Kelp Industrial Belt (Lazy Chainmail)

Not to be confused with Kelp Industrial Belt ({{aiNot Lazy Chainmail).}}
The kelp industrial belt is an item in the Minecraft mod Lazy Chainmail to run the Winding Jig. Kelp industrial belts can be crafted using 4 dried kelp (place kelp in the shape of a ring). The kelp industrial belt can be used to run the Winding Jig, using a durability every time an iron wire or spool is fully wound. Unbreaking decreases the chance that it takes damage, having a 50%, 33%, and 25% chance for levels I, II, and III respectively. Kelp industrial belts can be repaired through combining by crafting, using a grindstone, or using an anvil. Combining kelp industrial belts will add the two durability values together with an additional 3 durability. Combining will remove all enchantments, with some experience returned when combining in a grindstone. Combining in an anvil will keep and combine the enchantments on the industrial belts, costing experience. Kelp industrial belts can be enchanted with Unbreaking, Mending, and Curse of Vanishing using an enchanted book on an anvil. Kelp industrial belts cannot be enchanted in an enchanting table.
