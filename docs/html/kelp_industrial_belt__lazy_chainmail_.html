Kelp Industrial Belt (Lazy Chainmail)

Not to be confused with Kelp Industrial Belt ({{aiNot Lazy Chainmail).}}
<>The kelp industrial belt is an item (Minecraft) in the Minecraft mod (Minecraft) Lazy Chainmail to run the Winding Jig (Lazy Chainmail). Kelp industrial belts can be crafting (Minecraft) using 4 dried kelp (Minecraft) (place kelp in the shape of a ring). The kelp industrial belt can be used to run the Winding Jig, using a durability every time an iron wire (Lazy Chainmail) or iron spool (Lazy Chainmail) is fully wound. Unbreaking (Minecraft) decreases the chance that it takes damage, having a 50%, 33%, and 25% chance for levels I, II, and III respectively. Kelp industrial belts can be repaired through combining (Minecraft) by crafting (Minecraft), using a grindstone (Minecraft), or using an anvil (Minecraft). Combining kelp industrial belts will add the two durability values together with an additional 3 durability. Combining will remove all enchantments, with some experience (Minecraft) returned when combining in a grindstone. Combining in an anvil will keep and combine the enchantments on the industrial belts, costing experience. Kelp industrial belts can be enchanting (Minecraft) with Unbreaking (Minecraft), Mending (Minecraft), and Curse of Vanishing (Minecraft) using an enchanted book (Minecraft) on an anvil (Minecraft). Kelp industrial belts cannot be enchanted in an enchanting table.
