Kelp Industrial Belt (Not Lazy Chainmail)

<resistance|No||Speed|x1info>>The kelp industrial belt is an item in the Minecraft mod Not Lazy Chainmail and one of the eight industrial belts used to run machines. Kelp industrial belts can be crafted using 4 dried kelp (place kelp in the shape of a ring). Crafting the kelp industrial belt will award the advancement Slow But Steady. The chances to find a kelp industrial belt in a winding jig are the following:
In a winding jig ruins, there is a 28.57% chance to contain a kelp industrial belt. Kelp industrial belts are used in the following melding recipes:
Crude Engine - 1 kelp industrial belt, 1 block of iron, and 1 piston The kelp industrial belt can be used to run the following machines:
Auto Mailler
Auto Snipper
Auto Winder
Belt Generator
Chain Generator
Cobblestone Generator
Conveyor Warper
Crusher
Item Aggregator
Melder
Recycler
Winding Jig Kelp industrial belts can be repaired through combining by crafting, using a grindstone, or using an anvil. Combining kelp industrial belts will add the two durability values together with an additional 3 durability. Combining will remove all enchantments, with some experience returned when combining in a grindstone. Combining in an anvil will keep and combine the enchantments on the industrial belts, costing experience. Kelp industrial belts can be enchanted with Unbreaking, Mending, and Curse of Vanishing using an enchanted book on an anvil. Kelp industrial belts cannot be enchanted in an enchanting table. The rune Automated Permanence Rune can be cast on kelp industrial belts to give Automated Permanence (unbreakable), costing 2 levels. Kelp chainmail industrial belts can generate Forge Energy in a belt generator, generating 1 FE/t.
