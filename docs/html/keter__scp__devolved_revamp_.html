Keter (SCP: Devolved Revamp)

A Keter SCP is an NPC or object within SCP: Devolved Revamp with severe danger. Keter SCPs are a type of SCP that are extremely dangerous and harm players, usually killing them. They are terrible to use on Class Ds for tests without the proper care (besides their expendability). Staff personnel should make sure the test goes smoothly, avoiding casualties when possible.
