Knapped (Minecraft mod)

{{iKnapped}} is a Minecraft mod (Minecraft) made by Warm_Wooly in MCreator on December 18, 2024, adding the item (Minecraft) Knapped Flint Brick (Knapped) and new decoration Knapped blocks. Knapped is available for Minecraft: Java Edition 1.20.1 for Forge (Minecraft mod loader).<><> Knapped has an Minecraft mod addon for Create (Minecraft mod) compatibility, Create: Knapped.
