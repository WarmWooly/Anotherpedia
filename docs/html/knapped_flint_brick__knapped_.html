Knapped Flint Brick (Knapped)

For the block, see Knapped Flint Bricks ({{aiKnapped).}}&sp<>The knapped flint brick is an item in the Minecraft mod Knapped used in crafting the various blocks in the mod. Knapped flint bricks can be crafted with 2 flint (shapeless; makes 3). Knapped flint bricks can be cut from flint in a Stonecutter (makes 2).
