Knapping (Vintage Story)

Knapping is a crafting process (Vintage Story) within Vintage Story to create various early-game tool heads using stone (Vintage Story), flint (Vintage Story), or obsidian stone (Vintage Story). After selecting a recipe, voxel can be removed from the craft until it has been shaped to match the guidelines given. Knapping requires one stone to be placed on the ground, with a second one held to remove material.

While all stones can be used to remove material off a knapped item, only andesite (Vintage Story), basalt (Vintage Story), chert (Vintage Story), granite (Vintage Story), and peridotite (Vintage Story) are hard enough to be knapped into shape. Obsidian results in the strongest tools, followed by flint, then all other stones. The following items can be knapped:
Knapped Arrowhead (Vintage Story) x6
Knapped Axe Head (Vintage Story)
Knapped Hoe Head (Vintage Story)
Knapped Knife Blade (Vintage Story)
Shovel Head (Vintage Story)
Knapped Spear Head (Vintage Story)
