Large Canteen Table (Restaurant Tycoon 3)

<>The large canteen table (just the canteen table in-game) is a table (Restaurant Tycoon 3) in the Roblox game Restaurant Tycoon 3. A total of 6 customer (Restaurant Tycoon 3) can be seated normally at a large canteen table, with a max of 10 using optimal seating (Restaurant Tycoon 3). It is a part of the canteen set, which includes the canteen chair (Restaurant Tycoon 3) and the small canteen table (Restaurant Tycoon 3).
