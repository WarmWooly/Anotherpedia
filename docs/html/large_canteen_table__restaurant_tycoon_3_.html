Large Canteen Table (Restaurant Tycoon 3)

<>The large canteen table (just the canteen table in-game) is a table in the Roblox game Restaurant Tycoon 3. A total of 6 customers can be seated normally at a large canteen table, with a max of 10 using optimal seating. It is a part of the canteen set, which includes the chair and the small table.
