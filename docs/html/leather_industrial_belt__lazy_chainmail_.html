Leather Industrial Belt (Lazy Chainmail)

Not to be confused with Leather Industrial Belt ({{aiNot Lazy Chainmail).}}
The leather industrial belt is an item in the Minecraft mod Lazy Chainmail to run the Winding Jig. Leather industrial belts can be crafted using 4 leather (place leather in the shape of a ring). The leather industrial belt can be used to run the Winding Jig, using a durability every time an iron wire or spool is fully wound. Unbreaking decreases the chance that it takes damage, having a 50%, 33%, and 25% chance for levels I, II, and III respectively. Leather industrial belts can be repaired through combining by crafting, using a grindstone, or using an anvil. Combining leather industrial belts will add the two durability values together with an additional 3 durability. Combining will remove all enchantments, with some experience returned when combining in a grindstone. Combining in an anvil will keep and combine the enchantments on the industrial belts, costing experience. Leather industrial belts can be enchanted with Unbreaking, Mending, and Curse of Vanishing using an enchanted book on an anvil. Leather industrial belts cannot be enchanted in an enchanting table.
