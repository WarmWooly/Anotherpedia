Leather Padding (Not Lazy Chainmail)

<resistance|Noinfo>>Leather padding is an item in the Minecraft mod Not Lazy Chainmail, acting as the base for the other leather paddings. Leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving +0.25 Armor. Leather padding can be melded in a Melder using 1 leather and 1 wool. Apprentice-level leatherworkers have a 50% chance to trade 1 emerald for 1 leather padding. In a leathersmithing table, leather padding can be applied to chainmail armor or shields. Each leather padding adds +0.25 Armor, with a full set of armor with 12 leather paddings giving +3 Armor. Shields will only add Armor when held in the main hand or offhand.
