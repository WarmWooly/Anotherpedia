Leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +0.25 Armor (Minecraft). Leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather (Minecraft) and 1 wool (Minecraft). In a leathersmithing table (Not Lazy Chainmail), leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each leather padding adds +0.25 Armor (Minecraft), with a full set of armor with 12 leather paddings giving +3 Armor. Shields will only add Armor when held in the main hand (Minecraft) or offhand (Minecraft).
