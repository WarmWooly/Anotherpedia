Lightweight Leather Padding (Not Lazy Chainmail)

<>Lightweight leather padding is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the leather padding (Not Lazy Chainmail). Lightweight leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +0.2 Attack Speed (Minecraft). Lightweight leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 1 gold ingot (Minecraft). Journeyman-level leatherworker (Minecraft) have a 50% chance to trade 1 leather padding (Not Lazy Chainmail) and 4 emerald (Minecraft) for 1 lightweight leather padding. In a leathersmithing table (Not Lazy Chainmail), lightweight leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each lightweight leather padding adds +0.2 Attack Speed (Minecraft), with a full set of armor with 12 lightweight leather paddings giving +2.4 Attack Speed. Shields will only add Attack Speed when held in the main hand (Minecraft) or offhand (Minecraft).
