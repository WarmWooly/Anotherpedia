Lightweight Leather Padding (Not Lazy Chainmail)

<>Lightweight leather padding is an item in the Minecraft mod Not Lazy Chainmail as an upgrade to the leather padding. Lightweight leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving +0.2 Attack Speed. Lightweight leather padding can be melded in a Melder using 1 leather padding and 1 gold ingot. Journeyman-level leatherworkers have a 50% chance to trade 1 leather padding and 4 emeralds for 1 lightweight leather padding. In a leathersmithing table, lightweight leather padding can be applied to chainmail armor or shields. Each lightweight leather padding adds +0.2 Attack Speed, with a full set of armor with 12 lightweight leather paddings giving +2.4 Attack Speed. Shields will only add Attack Speed when held in the main hand or offhand.
