Loose Bedrock (Not Lazy Chainmail)

<>Loose bedrock is a block (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail that generates among bedrock (Minecraft) in the Overworld (Minecraft). Mining loose bedrock drops a bedrock shard (Not Lazy Chainmail). Loose bedrock can generate in bedrock (Minecraft) between y-level (Minecraft) -54 to -62. Bedrock only generates between -60 and -64, making the effective generation range between -60 and -62. When using a diamond pickaxe (Minecraft) or better, the loose bedrock can be mined. When mined, it will be converted back into bedrock (Minecraft) and drop a bedrock shard (Not Lazy Chainmail). Fortune (Minecraft) does not increase the number of dropped shards. Breaking loose bedrock with tools below diamond will turn the loose bedrock into bedrock, but it will not drop a bedrock shard.
