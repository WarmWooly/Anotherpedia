Loose Bedrock (Not Lazy Chainmail)

<>Loose bedrock is a block in the Minecraft mod Not Lazy Chainmail that generates among bedrock in the Overworld. Mining loose bedrock drops a bedrock shard. Loose bedrock can generate in bedrock between y-levels -54 to -62. Bedrock only generates between -60 and -64, making the effective generation range between -60 and -62. When using a diamond pickaxe or better, the loose bedrock can be mined. When mined, it will be converted back into bedrock and drop a bedrock shard. Fortune does not increase the number of dropped shards. Breaking loose bedrock with tools below diamond will turn the loose bedrock into bedrock, but it will not drop a bedrock shard.
