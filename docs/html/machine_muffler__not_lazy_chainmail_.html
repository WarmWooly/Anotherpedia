Machine Muffler (Not Lazy Chainmail)

For the block that reduces the sound, but doesn't mute it, see Machine Dampener ({{aiNot Lazy Chainmail).}}&sp<>The machine muffler is a block in the Minecraft mod Not Lazy Chainmail used to silence machines placed nearby. The machine muffler can be crafted with 1 wool and 8 cobblestone (place the wool in the center and surround with cobblestone). Crafting the machine muffler will award the advancement Peace and Quiet. Machines that are within 3 blocks of the machine muffler will be muted. This feature is most useful when creating automatic setups near a base, as the machines can be noisy.
