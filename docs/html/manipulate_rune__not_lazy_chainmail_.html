Manipulate Rune (Not Lazy Chainmail)

The manipulate rune is a rune in the Minecraft mod Not Lazy Chainmail. It is used to apply Manipulating to pliers, allowing additional blocks to be manipulated. The manipulate rune can be melded in a Melder using 1 iron composite nugget and 1 pseudorock. The manipulate rune can be cast onto pliers, using 2 levels. This is performed by holding the rune in the offhand and the item to cast in the main hand, then using the rune with the required levels, applying Manipulating. The manipulate rune will not cast if the item is already enchanted with Manipulating.
