Manipulate Rune (Not Lazy Chainmail)

<>The manipulate rune is a runes (Not Lazy Chainmail) in the Minecraft mod (Minecraft) Not Lazy Chainmail. It is used to apply Manipulating (Not Lazy Chainmail) to pliers (Not Lazy Chainmail), allowing additional block (Minecraft) to be plier block manipulation (Not Lazy Chainmail). The manipulate rune can be melded in a Melder (Not Lazy Chainmail) using 1 iron composite nugget (Not Lazy Chainmail) and 1 pseudorock (Not Lazy Chainmail). The manipulate rune can be cast onto pliers (Not Lazy Chainmail), using 2 experience (Minecraft). This is performed by holding the rune in the offhand (Minecraft) and the item to cast in the main hand (Minecraft), then using the rune with the required levels, applying Manipulating (Not Lazy Chainmail). The manipulate rune will not cast if the item is already enchanted with Manipulating or if the item is enchanted with Reversed (Not Lazy Chainmail).
