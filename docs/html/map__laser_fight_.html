Map (Laser Fight)

A {{bmap}} is a selectable battleground in {{iLaser Fight (Roblox game)}}. The map can be voted on between rounds, with the highest-voted map being loaded. If multiple maps are tied, then the leftmost map in the voting menu will be selected. There are a total of 13 maps, with 10 of them being able to be selected in normal play. 3 maps are "VIP maps" that can only be played on a private server (Roblox) through commands (Laser Fight).
