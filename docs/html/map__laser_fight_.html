Map ({{iLaser Fight}})

<><>A {{bmap}} is a selectable battleground in {{i[[Laser Fight|Laser Fight (Roblox game)]]}}. The map can be voted on between rounds, with the highest-voted map being loaded. If multiple maps are tied, then the leftmost map in the voting menu will be selected. There are a total of 13 maps, with 10 of them being able to be selected in normal play. 3 maps are "VIP maps" that can only be played on a [[private server|private server (Roblox)]] through [[commands|commands (Laser Fight)]].
