Marge'arita

The Marge'arita is an alcoholic beverage created by Rhyme x Reason Brewery. It prominently features The Simpsons character Marge Simpson. The name is a combination of Marge and Margarita. It has Montueka hops and Pacifica hops, using beech smoked malt as a malt and flavorings from habanero, salt, lime juice, orange zest, and monosodium glutamate.
