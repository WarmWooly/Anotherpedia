Marge'arita

The {{bMarge'arita}} is an alcoholic beverage created by Rhyme x Reason Brewery. It prominently features {{iThe Simpsons}} character Marge Simpson. The name is a combination of {{iMarge}} and {{iMargarita}}. It has Montueka hops and Pacifica hops, using beech smoked malt as a malt and flavorings from habanero, salt, lime juice, orange zest, and monosodium glutamate.
