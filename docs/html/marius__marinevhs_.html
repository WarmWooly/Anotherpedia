For the original rendition of Marius, see Marius (floriane_clabri).
Marius is an original character by floriane_clabri drawn by marinevhs for the #drawthisinyourstyle challenge from the original on Instagram on December 17, 2018. The post garnered over 8 thousand likes and multiple positive comments. Main page: Turtle (RGL.gg member)
On the website for the Team Fortress 2 league, RGL.gg, there is a user with the name "Turtle" who uses Marius as their profile picture. Their SteamId is 76561198289262787. In the Amino for Dead by Daylight, Marius (named "Matthew Eugene McLaughlin") is used as an original character for a survivor.
