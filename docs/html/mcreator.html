MCreator

MCreator is open-source software used for modding Minecraft, with built-in templates and a block-based programming system for creating new game elements based on preexisting mechanics (such as new blocks, items, and mobs). It can also be used for creating add-ons, resource packs, and data packs, with a built-in IDE for editing the code directly and a pixel art editor. MCreator is especially popular for beginners because of its simple interface and ease of access.
