Mekanism (Minecraft mod)

{{bMekanism}} is a Minecraft mod (Minecraft) made by bradyaidanc where Player (Minecraft) can create machine and mechanism to automation resource collection, Item (Minecraft) such as jetpack (Mekanism) to increase movement capabilities, and fluid (modded Minecraft), gas (modded Minecraft), and energy (modded Minecraft). Mekanism is available for Minecraft: Java Edition 1.20.1, Minecraft: Java Edition 1.19.2, Minecraft: Java Edition 1.19.1, Minecraft: Java Edition 1.19, Minecraft: Java Edition 1.18.2, Minecraft: Java Edition 1.16.5, Minecraft: Java Edition 1.16.4, Minecraft: Java Edition 1.16.3, Minecraft: Java Edition 1.16.1, Minecraft: Java Edition 1.15.2 Minecraft: Java Edition 1.12.2, Minecraft: Java Edition 1.12.1, Minecraft: Java Edition 1.11.2, Minecraft: Java Edition 1.10.2, and Minecraft: Java Edition 1.7.10. Main pages: Mekanism Additions (Minecraft mod), Mekanism Generators (Minecraft mod), and Mekanism Tools (Minecraft mod)&spThere are 3 mods that expand upon Mekanism: Mekanism Additions, Mekanism Generators, and Mekanism Tools. Mekanism Additions adds small general additions, such as baby mobs, balloons, and obsidian TNT. Mekanism Generators adds new single block and multiblock (modded Minecraft) generator, alongside new liquids and gasses. Mekanism Tools adds paxels and new tool materials, such as lapis lazuli (Minecraft) and osmium ingot (Mekanism).
