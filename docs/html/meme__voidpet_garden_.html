Meme (Voidpet Garden)

Meme is an action that a user can partake in Voidpet Garden where they create an image macro with a Voidpet. The message "Created with Voidpet" appears at the bottom of the created image. The action can be taken an unlimited number of times and can be immediately shared. The top message can be edited or randomized to a preset message. The size of the image vertically increases based on the length of the message.
