Memesis

Memesis, also titled Memesis Free and Memesis: Free Meme Generator is a mobile application for Android (operating system) used to create internet meme created by Kreutze Studios created in the mid-2010s. It offers various features for creating memes, including adding and overlapping image, applying text message with various font, and saving and sharing the final result.
