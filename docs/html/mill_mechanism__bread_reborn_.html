Mill Mechanism (Bread Reborn)

<resistance|Noinfo>>The mill mechanism is a block in the Minecraft mod Bread Reborn used to mill items into flour. A mill mechanism naturally generates in a mill. To use the mill mechanism, the item to be processed must be held in the main hand and a stick must be held in the offhand.
