Mill Mechanism (Bread Reborn)

<>The mill mechanism is a block (Minecraft) in the Minecraft mod (Minecraft) Bread Reborn used to mill items into flour. A mill mechanism naturally generates in a mill (Bread Reborn). To use the mill mechanism, the item to be processed must be held in the main hand (Minecraft) and a stick (Minecraft) must be held in the offhand (Minecraft).
