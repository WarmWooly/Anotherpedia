Mindustry

Mindustry is an indie game developed by Anuken. It was made in April 2017 for the 2017 GDL Metal Monstrosity Jam, which won first place on the 30th. It was made in LibGDX, programmed in Java. Mindustry doesn't fall well into 1-2 distinct genres, taking elements from RTS, sandbox, and tower defense. It also has a dedicated multiplayer scene and modding scene which expands the singleplayer campaign. <>, it is on update 7.0, build 146. It can be played on mobile devices and computers for free, or purchased on Steam for $9.99.
