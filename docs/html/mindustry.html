Mindustry

{{i{{bMindustry}}}} is an indie game development by Anuken. It was made in April 2017 for the 2017 GDL Metal Monstrosity Jam, which winning first on the 30th. It was made in LibGDX, programming in Java (programming language). {{iMindustry}} doesn't fall well into 1-2 distinct video game genre, taking elements from Real-time strategy game, sandbox game, and tower defense game. It also has a dedicated Mindustry server and Mindustry modding which expands the singleplayer Mindustry campaign. <>, it is on Mindustry updates 7.0 (Mindustry), Mindustry builds Build 146 (Mindustry). It can be played on mobile device and computer for free, or purchase on Steam (platform) for $9.99.
