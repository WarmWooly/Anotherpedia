Minecraftle

{{bMinecraftle}} is a browser game Wordle-like for crafting recipe (Minecraft) from Minecraft developed by Tamura Boog, Zach Manson, Harrison Oates, and Ivan Sossa Gongora in 2022, hosted at <>. The game has the player make a crafting recipe in a crafting table (Minecraft) using the 18 item (Minecraft) given. A random recipe is given, so the player must figure out, in 10 recipe guesses, what the correct recipe is. If the item is not used in the recipe, it is grayed out. If it is used in the recipe, but misplaced, it is colored yellow, and if it is placed in the correct spot, it is colored green. The player wins after guessing the correct recipe. There is a daily game and a random game option.

A total of 18 items can be used to make recipes: wooden planks (Minecraft), cobblestone (Minecraft), stone (Minecraft), glass (Minecraft), wool (Minecraft), stick (Minecraft), coal (Minecraft), diamond (Minecraft), gold ingot (Minecraft), iron ingot (Minecraft), redstone dust (Minecraft), nether quartz (Minecraft), wooden slab (Minecraft), log (Minecraft), iron nugget (Minecraft), redstone torch (Minecraft), string (Minecraft), and leather (Minecraft). Only valid recipes may be input as solutions.
