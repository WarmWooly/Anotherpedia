Muddy Leather Padding (Not Lazy Chainmail)

<><>||{{bStats}}||{{b[[Rarity|Rarity (Minecraft)]]}}|Common||{{b[[Stack size|Stack size (Minecraft)]]}}|64||{{b[[Fire/lava&spresistance|Fire-resistant item (Minecraft)]]}}|Noinfo>>{{bMuddy leather padding}} is an [[item|item (Minecraft)]] in the {{i[[Minecraft]]}} [[mod|mod (Minecraft)]] {{i[[Not Lazy Chainmail]]}} as an upgrade to the [[leather padding|leather padding (Not Lazy Chainmail)]]. Muddy leather padding is used to upgrade [[chainmail armor|chainmail armor (Not Lazy Chainmail)]] or [[shields|shield (Minecraft)]] in a [[leathersmithing table|leathersmithing table (Not Lazy Chainmail)]], giving +1 Muddiness and -0.01 [[Movement Speed|Movement Speed (Minecraft)]]. Muddiness causes mobs hit by the wearer to be given [[Blindness|Blindness (Minecraft)]] for 2 * Muddiness seconds.<><>Muddy leather padding can be melded in a [[Melder|Melder (Not Lazy Chainmail)]] using 1 [[leather padding|leather padding (Not Lazy Chainmail)]] and 2 [[mudballs|mudball (Not Lazy Chainmail)]].<><>In a [[leathersmithing table|leathersmithing table (Not Lazy Chainmail)]], muddy leather padding can be applied to [[chainmail armor|chainmail armor (Not Lazy Chainmail)]] or [[shields|shield (Minecraft)]]. Each muddy leather padding adds +1 Muddiness and -0.01 Movement Speed, with a full set of armor with 12 muddy leather paddings giving +12 Muddiness and -0.12 Movement Speed. Shields will only add Muddiness and remove Movement Speed when held in the [[main hand|main hand (Minecraft)]] or [[offhand|offhand (Minecraft)]].
