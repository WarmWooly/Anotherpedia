Muddy Leather Padding (Not Lazy Chainmail)

<>Muddy leather padding is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the leather padding (Not Lazy Chainmail). Muddy leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +1 Muddiness and -0.01 Movement Speed (Minecraft). Muddiness causes mobs hit by the wearer to be given Blindness (Minecraft) for 2 * Muddiness seconds. Muddy leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 2 mudball (Not Lazy Chainmail). In a leathersmithing table (Not Lazy Chainmail), muddy leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each muddy leather padding adds +1 Muddiness and -0.01 Movement Speed, with a full set of armor with 12 muddy leather paddings giving +12 Muddiness and -0.12 Movement Speed. Shields will only add Muddiness and remove Movement Speed when held in the main hand (Minecraft) or offhand (Minecraft).
