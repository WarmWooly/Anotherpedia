Muffle Rune (Not Lazy Chainmail)

The muffle rune is a rune in the Minecraft mod Not Lazy Chainmail. It is used to apply Muffled to Not Lazy Chainmail tools. The muffle rune can be melded in a Melder using 1 machine muffler and 1 smooth stone. The muffle rune can be cast onto a chainsaw, chainsword, chain launcher, chainmail project, sieve mesh, pliers, rune scraper, whip, flail, nunchucks, or lasso, using 1 level. This is performed by holding the rune in the offhand and the item to cast in the main hand, then using the rune with the required levels, applying Muffled. The muffled rune will not cast if the item is already enchanted with Muffled or if the item is enchanted with Dampened.
