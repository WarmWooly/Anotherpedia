Muffle Rune (Not Lazy Chainmail)

<>The muffle rune is a runes (Not Lazy Chainmail) in the Minecraft mod (Minecraft) Not Lazy Chainmail. It is used to apply muffled (Not Lazy Chainmail) to Not Lazy Chainmail tools. The muffle rune can be melded in a Melder (Not Lazy Chainmail) using 1 machine muffler (Not Lazy Chainmail) and 1 pseudorock (Not Lazy Chainmail). The muffle rune can be cast onto a chainsaw (Not Lazy Chainmail), chainsword (Not Lazy Chainmail), chain launcher (Not Lazy Chainmail), chainmail project (Not Lazy Chainmail), sieve meshes (Not Lazy Chainmail), pliers (Not Lazy Chainmail), rune scraper (Not Lazy Chainmail), whips (Not Lazy Chainmail), flails (Not Lazy Chainmail), nunchucks (Not Lazy Chainmail), or lassos (Not Lazy Chainmail), using 1 experience (Minecraft). This is performed by holding the rune in the offhand (Minecraft) and the item to cast in the main hand (Minecraft), then using the rune with the required levels, applying muffled (Not Lazy Chainmail). The muffled rune will not cast if the item is already enchanted with Muffled or if the item is enchanted with Dampened (Not Lazy Chainmail).
