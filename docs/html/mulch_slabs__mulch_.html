Mulch slabs (Mulch)

<>||{{bFire-resistant item (Minecraft)}}|Noinfo>>{{bMulch slabs}} are a type of falling block (Minecraft) slab (Minecraft) in the {{iMinecraft}} mod (Minecraft) {{iMulch (Minecraft mod)}} used as decoration among full mulch (mulch) blocks. All mulch slabs can be crafting (Minecraft) by combining of the same type of wood chips (mulch) in a 2-wide line or by placing 3 of the same mulch (Mulch) in a 3-wide line (makes 6). 2 of the same mulch slabs can be placed in a vertical line to be recombined into mulch (Mulch) of the same type. All mulch slabs, besides Crimson, Warped, and Hellfire, can be used as fuel (Minecraft) in a furnace (Minecraft), smelting 1.5 items per block. All mulch slabs can be thrown into a wood chipper (Mulch) to be turned into 2 wood chips (Mulch) of the same type. All mulch slabs can be thrown into a mulch redefiner (mulch) to be turned into all non-dyed mulch slabs.
