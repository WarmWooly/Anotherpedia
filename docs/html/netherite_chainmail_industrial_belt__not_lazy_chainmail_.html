Netherite chainmail industrial belts can be repaired through combining (Minecraft) by crafting (Minecraft), using a grindstone (Minecraft), or using an anvil (Minecraft). Combining netherite chainmail industrial belts will add the two durability values together with an additional 3 durability. Combining will remove all enchantments, with some experience (Minecraft) returned when combining in a grindstone. Combining in an anvil will keep and combine the enchantments on the industrial belts, costing experience. Netherite chainmail industrial belts can be enchanting (Minecraft) with Unbreaking (Minecraft), Mending (Minecraft), and Curse of Vanishing (Minecraft) using an enchanted book (Minecraft) on an anvil (Minecraft). Netherite chainmail industrial belts cannot be enchanted in an enchanting table.
