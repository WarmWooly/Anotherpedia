Netherite Ornate Lamp (Not Lazy Chainmail)

<resistance|Yesinfo>>The netherite ornate lamp is a block in the Minecraft mod Not Lazy Chainmail used as redstone-powered lighting. The netherite ornate lamp can be crafted with 2 netherite chain carpets and 1 redstone lamp (place the lamp in the middle, then the carpets above and below). When powered, the netherite ornate lamp produces light. Unlike redstone lamps, the netherite ornate lamp takes 0 redstone ticks to turn off. gal
