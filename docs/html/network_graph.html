Network graph

<><>A {{bnetwork graph}}, also known as a {{bnetwork diagram}} or {{bnetwork chart}}, is a [[graph]] that shows [[connections|connection]] between [[data]]. A network graph is composed of [[nodes|node]] and [[lines|line]] or [[arrows|arrow]] which show the [[relationship]] between two data points. Many network graphs change the size of the nodes to be larger with more connections and smaller with fewer.
