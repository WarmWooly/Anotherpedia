Nintendo Switch Afterglow Wireless Controller

The Nintendo Switch Afterglow Wireless Controller is a wireless game controller for the Nintendo Switch produced by Performance Designed Products (PDP). <>, the controller costs $49.99. The controller has a clear shell with LEDs inside, which can be programmed to emit various colors. A white Nintendo Switch logo is plastered on the front, with all the normal controls of a Pro Controller.
