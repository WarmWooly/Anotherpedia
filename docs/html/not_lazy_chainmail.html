Not Lazy Chainmail

{{iNot Lazy Chainmail}} is a Minecraft mod (Minecraft) made by Warm_Wooly in MCreator on July 23, 2024, giving an alternative method to obtain Chainmail (Minecraft), alongside new Not Lazy Chainmail items, Not Lazy Chainmail blocks, and Not Lazy Chainmail advancements. Not Lazy Chainmail is available for Minecraft: Java Edition 1.20.1 for Forge (Minecraft mod loader).<><> Not Lazy Chainmail requires Curios API. The alternative mod, Lazy Chainmail, is a bare-bones version of Not Lazy Chainmail.
