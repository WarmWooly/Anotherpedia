Not Lazy Chainmail 1.0.0

{{iNot Lazy Chainmail 1.0.0}} was the full release to the Minecraft mod (Minecraft) Not Lazy Chainmail on July 23rd, 2024. The version features the majority of mechanics for the mod, including Winding Jig (Not Lazy Chainmail), snipping (Not Lazy Chainmail), and Chainmail Project (Not Lazy Chainmail). It also features most of the current decorative block (Minecraft), weapon (Minecraft), and automation-related items. The first Not Lazy Chainmail updates to the mod would come two days later as Not Lazy Chainmail 1.0.1.
