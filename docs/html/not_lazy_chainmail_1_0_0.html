Not Lazy Chainmail 1.0.0

{{iNot Lazy Chainmail 1.0.0}} was the full release to the Minecraft mod Not Lazy Chainmail on July 23rd, 2024. The version features the majority of mechanics for the mod, including winding, snipping, and maille crafting. It also features most of the current decorative blocks, weapons, and automation-related items. The first update to the mod would come two days later as 1.0.1.
