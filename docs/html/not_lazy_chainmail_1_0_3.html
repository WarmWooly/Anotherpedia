Not Lazy Chainmail 1.0.3

Not Lazy Chainmail 1.0.2 was a minor update to the Minecraft mod Not Lazy Chainmail released on August 23rd, 2024, which updates the recipe for the conveyor manager and balances sieving drops. This update follows 1.0.2 and precedes the Halloween Update (1.1.0).
