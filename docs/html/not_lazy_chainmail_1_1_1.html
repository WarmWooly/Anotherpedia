Not Lazy Chainmail 1.1.1

{{iNot Lazy Chainmail 1.1.1}} was a minor update to the Minecraft mod Not Lazy Chainmail released on November 23rd, 2024, which expanded hopper dropper functionality and gave some items the ability to be used as trims. This update follows the Halloween Update (1.0.0) and precedes 1.1.2. These are all the items and features that were fixed, balanced, or updated:;;The hopper dropper can now be used on chests, trapped chests, barrels, shulker boxes, furnaces, smokers, blast furnaces, and crushers.|The rose gold ingot, iron composite ingot, telemagnetic alloy, and chromatic mass can now be used as armor trims.|Fixed a typo in the advancement Cobblestone is Everything.;;
