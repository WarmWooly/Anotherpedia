NUMBER GO UP (The Stupendium song)

"NUMBER GO UP" is a song produced by The Stupendium released onto Spotify, iTunes, and YouTube on September 5, 2025 with an a cappella released the same day. The song, with an accomanying music video, is vaugely based on the game Balatro, utilizing the imagry, naming, and designs of multiple Jokers present within the game while creating new ones for the video. It also ties itself close to the themes of gambling, addiction, and the validation of a number going up.
