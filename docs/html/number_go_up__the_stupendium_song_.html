NUMBER GO UP (The Stupendium song)

{{b"NUMBER GO UP"}} is a song produced by The Stupendium released onto Spotify, iTunes, and YouTube on September 5, 2025 with an NUMBER GO UP - A Cappella released the same day.<><><> The song, with an accomanying music video, is vaugely based on the video game {{iBalatro}}, utilizing the imagry, naming, and designs of multiple Jokers (Balatro) present within the game while creating new ones for the video. It also ties itself close to the themes of gambling, addiction, and the validation of a number go up. The music beat was produced by SINIMA BEATS, with the saxaphone preformed by CK9C and the song song mixing by OxygenBeats.
