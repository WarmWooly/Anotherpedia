One Gun Run (Laser Fight)

One Gun Run is a mode (Laser Fight) in Laser Fight (Roblox game) where the player spawns in with a single random gun (Laser Fight). This is similar to the mode Starting Weapons (Laser Fight), where players spawn with 3 random guns instead of 1. Guns will not gun spawning (Laser Fight) and can't be dropped in this mode.
