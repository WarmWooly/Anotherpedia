Order sequence

Order sequence in mathematics is a non-decreasing sequence of non-negative integer representing an algrebraic structure based on the order (group theory) of subgroup generated set of a group from each element (group theory) in the group. The order sequence takes inspiration from the degree sequence in graph theory, with the same formatting and element-based determination. The order sequence can be used to determine if two algebraic structures are not group isomorphism to each other, as they must have the same order sequence to be isomorphic. Matching order sequences between systems does not imply isomorphism. The order sequence of the structure [&bbZs-b3; +s-b3] (where &bbZs-b3 = {0, 1, 2}, and +s-b3 is the operation (mathematics) "(a + b) modulo 3") would be determined with the following generated subgroups:

⟨0⟩ = {0}; ord(⟨0⟩) = 1
⟨1⟩ = {0, 1, 2}; ord(⟨1⟩) = 3
⟨2⟩ = {0, 1, 2}; ord(⟨2⟩) = 3

From that, the order sequence that would describe [&bbZs-b3; +s-b3] is (1, 3, 3). Note that despite the order of the generated subgroups matching up with the final order of the sequence, this is not always the case. The elements in the sequence should always be numbered from the least to greatest.
