Phase transition

<><>A {{bphase transition}} is when [[matter]] changes from one [[state|states of matter]] (such as a [[solid]]) into a different state of matter (such as a [[liquid]]). The phase transitions are:<>;;Solid --> Liquid = [[Melting]]|Liquid --> [[Gas]] = [[Vaporization]]|Solid --> Gas = [[Sulimation]]|Gas --> [[Plasma]] = [[Ionization]]|Liquid --> Solid = [[Freezing]]|Gas --> Liquid = [[Condensation]]|Gas --> Solid = [[Deposition]]|Plasma --> Gas = [[Deionization]];;
