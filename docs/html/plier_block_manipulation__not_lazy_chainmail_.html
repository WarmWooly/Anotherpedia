Plier block manipulation (Not Lazy Chainmail)

Plier block manipulation is a mechanic in the Minecraft Minecraft modding Not Lazy Chainmail where pliers (Not Lazy Chainmail) can be used on various block (Minecraft) to instantly mining (Minecraft) them or block rotation (Minecraft) them. Each use costs 1 durability (Minecraft), affected by Unbreaking (Minecraft) and Automated Permanence (Not Lazy Chainmail).
