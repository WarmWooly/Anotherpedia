Plier block manipulation (Not Lazy Chainmail)

Plier block manipulation is a mechanic in the Minecraft mod Not Lazy Chainmail where pliers can be used on various blocks to instantly break them or rotate them. Pliers can be used to rotate blocks. Blocks will rotate clockwise, and with Reversed, they rotate counterclockwise.
