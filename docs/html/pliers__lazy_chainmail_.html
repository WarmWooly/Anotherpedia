Pliers (Lazy Chainmail)

Not to be confused with Pliers ({{aiNot Lazy Chainmail) or Iron Pliers (aiNot Lazy Chainmail).}}
<>Pliers are an item (Minecraft) in the Minecraft mod (Minecraft) Lazy Chainmail used for snipping iron chain link (Lazy Chainmail) from iron coil (Lazy Chainmail) and unfinished project (Lazy Chainmail). Pliers can be crafting (Minecraft) with 2 iron ingot (Minecraft), 1 planks (Minecraft), and 2 stick (Minecraft) (place the planks in the middle, then the iron above and to the right, and the sticks to the bottom and the left). Hold the pliers in the offhand (Minecraft) with the iron coil or unfinished project to be snipped in the main hand (Minecraft). To remove a chain from the coil/unfinished project, right-click anywhere. This will use the pliers and create one iron chain link. When a coil/unfinished project goes down to 0 remaining chains, it will be removed. Pliers can be used on chain (Minecraft) to instantly harvest them, taking a point of durability, using the same Unbreaking rules as above. Pliers are ideally not used in combat. Each attack will cost 2 durability. Pliers deal 2 damage and have an attack cooldown of 1 (1 second or 20 tick (Minecraft)). Pliers can be repaired through combining (Minecraft) by crafting (Minecraft), using a grindstone (Minecraft), or using an anvil (Minecraft). Combining pliers will add the two durability values together with an additional 3 durability. Combining will remove all enchantments, with some experience (Minecraft) returned when combining in a grindstone. Combining in an anvil will keep and combine the enchantments on the pliers, costing experience. Pliers can be enchanting (Minecraft) with Unbreaking (Minecraft), Mending (Minecraft), and Curse of Vanishing (Minecraft), using an enchanted book (Minecraft) on an anvil (Minecraft). Unbreaking may also be obtained in an enchanting table (Minecraft).
