Pluto (Balatro)

Pluto is a Planet Cards (Balatro) in Balatro based on the dwarf planet Pluto. When used, it levels up the poker hands (Balatro), adding 10 chips (Balatro) and 1 multiplier (Balatro) when a High Card is played. Pluto is available at the start of the game. If all Planet Cards are held while generating a random Planet Card, it will select Pluto (ex. using Perkeo (Balatro) to create a Negative (Balatro) copy of every Planet Card will allow this to happen).
