Pluto (Balatro)

Pluto is a Planet Card in Balatro based on the dwarf planet Pluto. When used, it levels up the High Card, adding 10 chips and 1 mult when a High Card is played. Pluto is available at the start of the game. If all Planet Cards are held while generating a random Planet Card, it will select Pluto (ex. using Perkeo to create a Negative copy of every Planet Card will allow this to happen).
