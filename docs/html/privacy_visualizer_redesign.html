Privacy Visualizer Redesign

The Privacy Visualizer Redesign is a redesigned version of Reject Convenience's Privacy Visualizer browser tool made by rodak123, hosted at <>. It takes the same categories and UI layout of the Privacy Visualizer, but streamlines it in multiple areas, including being more responsive to user input, being revamped with Bootstrap (front-end framework), and having the URL dynamically update its URL query based on the selected inputs to be saved and shared with others. The redesign is directly mentioned by Reject Convenience on his website.
