Programming language

A programming language is a method of turning Character (computer programming) into instruction for an electronic device to complete a task. Someone who writes with a programming language is known as a programmer. There are multiple types of programming languages, such as: A procedural programming language is a programming language where each step is done in sequential order. These include ALGOL (programming language), COBOL, and FORTRAN. An object-oriented programming language is a programming language where there are defined Object (computer programming) with Attribute (computer programming). These include Java, Python (programming language), and C++. A functional programming language is a programming language where everything is defined by mathematics Function (computer programming). These include Haskell, Lisp (programming language), and Clojure.
