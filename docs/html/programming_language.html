Programming language

A programming language is a method of turning characters into instructions for an electronic device to complete a task. Someone who writes with a programming language is known as a programmer. There are multiple types of programming languages, such as: A procedural programming language is a programming language where each step is done in sequential order. These include ALGOL, COBOL, and FORTRAN. An object-oriented programming language is a programming language where there are defined objects with attributes. These include Java, Python, and C++. A functional programming language is a programming language where everything is defined by mathematical functions. These include Haskell, Lisp, and Clojure.
