Reinforced Leather Padding (Not Lazy Chainmail)

<resistance|Yesinfo>>Reinforced leather padding is an item in the Minecraft mod Not Lazy Chainmail as an upgrade to the leather padding. Reinforced leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving +1 Knockback Resistance. Reinforced leather padding can be melded in a Melder using 1 leather padding and 1 netherite scrap. Master-level leatherworkers have a 50% chance to trade 1 leather padding and 12 emeralds for 1 reinforced leather padding. In a leathersmithing table, reinforced leather padding can be applied to chainmail armor or shields. Each reinforced leather padding adds +1 Knockback Resistance, with a full set of armor with 12 reinforced leather paddings giving +12 Knockback Resistance. Shields will only add Knockback Resistance when held in the main hand or offhand.
