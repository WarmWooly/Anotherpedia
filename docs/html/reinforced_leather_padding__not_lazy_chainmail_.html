Reinforced Leather Padding (Not Lazy Chainmail)

<>Reinforced leather padding is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the leather padding (Not Lazy Chainmail). Reinforced leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +1 Knockback Resistance (Minecraft). Reinforced leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 1 netherite scrap (Minecraft). Master-level leatherworker (Minecraft) have a 50% chance to trade 1 leather padding (Not Lazy Chainmail) and 12 emerald (Minecraft) for 1 reinforced leather padding. In a leathersmithing table (Not Lazy Chainmail), reinforced leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each reinforced leather padding adds +1 Knockback Resistance (Minecraft), with a full set of armor with 12 reinforced leather paddings giving +12 Knockback Resistance. Shields will only add Knockback Resistance when held in the main hand (Minecraft) or offhand (Minecraft).
