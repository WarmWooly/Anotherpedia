Reverse Rune (Not Lazy Chainmail)

The reverse rune is a rune in the Minecraft mod Not Lazy Chainmail. It is used to apply Reversed to pliers, making block rotation go counterclockwise. The reverse rune can be melded in a Melder using 1 rose gold nugget and 1 smooth stone. The reverse rune can be cast onto pliers, using 1 level. This is performed by holding the rune in the offhand and the item to cast in the main hand, then using the rune with the required levels, applying Reversed. The reverse rune will not cast if the item is already enchanted with Reversed.
