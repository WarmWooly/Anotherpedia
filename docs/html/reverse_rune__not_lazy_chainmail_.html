Reverse Rune (Not Lazy Chainmail)

<>The reverse rune is a runes (Not Lazy Chainmail) in the Minecraft mod (Minecraft) Not Lazy Chainmail. It is used to apply reversed (Not Lazy Chainmail) to pliers (Not Lazy Chainmail), making plier block manipulation (Not Lazy Chainmail) go counterclockwise. The reverse rune can be melded in a Melder (Not Lazy Chainmail) using 1 rose gold nugget (Not Lazy Chainmail) and 1 smooth stone (Minecraft). The reverse rune can be cast onto pliers (Not Lazy Chainmail), using 1 experience (Minecraft). This is performed by holding the rune in the offhand (Minecraft) and the item to cast in the main hand (Minecraft), then using the rune with the required levels, applying reversed (Not Lazy Chainmail). The reverse rune will not cast if the item is already enchanted with Reversed or if the item is enchanted with Manipulating (Not Lazy Chainmail).
