Rocky Leather Padding (Not Lazy Chainmail)

<>Rocky leather padding is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the leather padding (Not Lazy Chainmail). Rocky leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +1 Rockiness. Rockiness proportionally increases damage from Chain Launcher (Not Lazy Chainmail) Loose Stone (Not Lazy Chainmail)/Loose Deepened Stone (Not Lazy Chainmail). Rocky leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 2 loose deepened stone (Not Lazy Chainmail). In a leathersmithing table (Not Lazy Chainmail), rocky leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each rocky leather padding adds +1 Rockiness, with a full set of armor with 12 rocky leather paddings giving +12 Rockiness. Shields will only add Rockiness when held in the main hand (Minecraft) or offhand (Minecraft).
