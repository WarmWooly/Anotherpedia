Rocky Leather Padding (Not Lazy Chainmail)

<resistance|Noinfo>>Rocky leather padding is an item in the Minecraft mod Not Lazy Chainmail as an upgrade to the leather padding. Rocky leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving +1 Rockiness. Rockiness proportionally increases damage from lauched Loose Stones/Loose Deepened Stones. Rocky leather padding can be melded in a Melder using 1 leather padding and 2 loose deepened stones. In a leathersmithing table, rocky leather padding can be applied to chainmail armor or shields. Each rocky leather padding adds +1 Rockiness, with a full set of armor with 12 rocky leather paddings giving +12 Rockiness. Shields will only add Rockiness when held in the main hand or offhand.
