Rune Scraper (Not Lazy Chainmail)

The rune scraper is an item (technically a sword) in the Minecraft mod Not Lazy Chainmail, which can be used to remove rune enchantments. The rune scraper can be crafted with 1 bedrock shard and 1 rose gold ingot (place the ingot in the middle, then the shard to the bottom-left). When held in the offhand, the rune scraper can be used to remove rune enchantments from a held item, giving back experience based on the enchantments. If a rune enchantment is scraped off a rune scraper, the advancement Scraper Paradox will be awarded. Rune scrapers can be used to mine bamboo, bamboo saplings, cobwebs, and other blocks faster than other tools, costing 2 durability per block broken. In Creative, the rune scraper will not break blocks, but will destroy placed entities. While not specifically designed for combat, the rune scraper can technically be used, with each attack costing 1 durability. The rune scraper deals 1.5 damage and has an attack cooldown of 2 (0.5 seconds or 10 ticks). The rune scraper can be used to perform sweeping attacks like other swords. The rune scraper can be repaired through combining by crafting, using a grindstone, or using an anvil. Combining rune scrapers will add the two durability values together with an additional 3 durability. Combining will remove all enchantments, with some experience returned when combining in a grindstone. Combining in an anvil will keep and combine the enchantments on the rune scrapers, costing experience. Rune scrapers can be repaired in an anvil using a rose gold ingot or bedrock shard, restoring 25% of the rune scraper's durability, rounded down. Rune scrapers can be enchanted with Unbreaking, Mending, Curse of Vanishing, Sharpness, Smite, Bane of Arthropods, Knockback, Fire Aspect, Looting, and Sweeping Edge using an enchanted book on an anvil.
