Rune Scraper (Not Lazy Chainmail)

<>The chainsword is an item (Minecraft) (technically a sword (Minecraft)) in the Minecraft mod (Minecraft) Not Lazy Chainmail, which can be used to remove runes (Not Lazy Chainmail) enchantment (Minecraft). The rune scraper can be crafting (Minecraft) with 1 bedrock shard (Not Lazy Chainmail) and 1 rose gold ingot (Not Lazy Chainmail) (place the ingot in the middle, then the shard to the bottom-left). When held in the offhand (Minecraft), the rune scraper can be used to remove runes (Not Lazy Chainmail) enchantment (Minecraft) from a held item, giving back experience (Minecraft) based on the enchantments. If a rune enchantment is scraped off a rune scraper, the advancement (Minecraft) Scraper Paradox (Not Lazy Chainmail) will be awarded. Rune scrapers can be used to mine bamboo (Minecraft), bamboo sapling (Minecraft), cobweb (Minecraft), and other blocks faster than other tools, costing 2 durability per block broken. In Creative (Minecraft), the rune scraper will not break blocks, but will destroy placed entity (Minecraft). While not specifically designed for combat, the rune scraper can technically be used, with each attack costing 1 durability. The rune scraper deals 1.5 damage and has an attack cooldown of 2 (0.5 seconds or 10 tick (Minecraft)). The rune scraper can be used to perform Sweep attack (Minecraft) like other swords. The rune scraper can be repaired through combining (Minecraft) by crafting (Minecraft), using a grindstone (Minecraft), or using an anvil (Minecraft). Combining rune scrapers will add the two durability values together with an additional 3 durability. Combining will remove all enchantments, with some experience (Minecraft) returned when combining in a grindstone. Combining in an anvil will keep and combine the enchantments on the rune scrapers, costing experience. Rune scrapers can be repaired in an anvil using a rose gold ingot (Not Lazy Chainmail) or bedrock shard (Not Lazy Chainmail), restoring 25% of the rune scraper's durability, rounded down. Rune scrapers can be enchanting (Minecraft) with Unbreaking (Minecraft), Mending (Minecraft), Curse of Vanishing (Minecraft), Sharpness (Minecraft), Smite (Minecraft), Bane of Arthropods (Minecraft), Knockback (Minecraft), Fire Aspect (Minecraft), Looting (Minecraft), and Sweeping Edge (Minecraft) using an enchanted book (Minecraft) on an anvil (Minecraft). Unbreaking, Sharpness, Smite, Bane of Arthropods, Knockback, Fire Aspect, Looting, and Sweeping Edge may also be obtained in an enchanting table (Minecraft).
