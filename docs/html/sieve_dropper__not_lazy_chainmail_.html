Sieve Dropper (Not Lazy Chainmail)

<>The sieve dropper is a block (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail used to sieving (Not Lazy Chainmail) item (Minecraft entity) on chain conveyors (Not Lazy Chainmail). The sieve dropper can be crafting (Minecraft) with 1 sort dropper (Not Lazy Chainmail) and 4 block of iron chains (Not Lazy Chainmail) (place the dropper in the middle, surround with chains). Main page: Sieving ({{aiNot Lazy Chainmail)}}
Sieve droppers require a chain conveyor to be placed above them before they will detect items flowing over. They take items that flow above the sieve dropper, determine if the item meets the set criteria, and then sieve it below if the item is the one being checked for. Right-clicking the sieve dropper will open up its menu. Sieve droppers have 9 slots that can each hold an item to be sieved. If the Filter Type is set to "Whitelist", it will sieve items that match one of the items in the slots. If the Filter Type is set to "Blacklist", it will sieve items that do not match any of the items in the slots. There is an additional slot to place a sieve meshes (Not Lazy Chainmail). Automatically sieving uses durability (Minecraft) of the sieve mesh. Particles can be disabled using the "Particles" button. If a player is standing under a sieve dropper when it is producing experience (Not Lazy Chainmail) (by sieving sculk (Minecraft) with an iron sieve mesh (Not Lazy Chainmail)), they will be awarded the advancement (Minecraft) Shower, NOW!
