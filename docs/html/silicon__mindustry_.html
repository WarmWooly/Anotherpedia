Silicon (Mindustry)

{{bSilicon}} is an item (Mindustry) in {{iMindustry}} on Serpulo (Mindustry) and Erekir (Mindustry) that can be produced in a Silicon Smelter (Mindustry) or Silicon Crucible (Mindustry) using Sand (Mindustry) and coal (Mindustry) or Sand, Coal and Pyratite (Mindustry) respectively on Serpulo. For Erekir, Silicon can be produced in a Silicon Arc Furnace (Mindustry) using Sand and Graphite (Mindustry).
