Sliding (SCP: Devolved Revamp)

Sliding is an action that can be performed by Chaos Insurgency (SCP: Devolved Revamp) or Mobile Task Force (SCP: Devolved Revamp) members in SCP: Devolved Revamp which is triggered by pressing codeZ. The player must be on the ground to perform a slide, which will cause them to be jolted forward. Afterwards, a short cooldown will prevent additional slides.
