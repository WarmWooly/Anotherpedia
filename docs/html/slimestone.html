Slimestone

{{tFor the {{aiGoBots (franchise)}} character, see Slimestone ({{aiGoBots}}).}}&sp{{bSlimestone}} is a fan term for redstone contraption utilizing slime block (Minecraft)/honey block (Minecraft), sticky piston (Minecraft), and observer (Minecraft) to create motion in {{iMinecraft}}. Slime and honey blocks can drag various other block (Minecraft) and can be pushed by piston (Minecraft), allowing for machines that automatically update and push themselves along. Flying machine (Minecraft), automatic farm (Minecraft), redstone door (Minecraft), and other contraptions can be made or optimized utilizing slimestone.
