Smooth Quartz Flushwork Panel (Knapped)

<resistance|Noinfo>>The smooth quartz flushwork Panel is a smooth quartz flushwork block in the Minecraft mod Knapped. The smooth quartz flushwork Panel can be crafted with 1 smooth quartz block and 2 knapped flint bricks (item) (shapeless). The smooth quartz flushwork Panel can be cut from any other smooth quartz flushwork (besides Thing) in a Stonecutter. The smooth quartz flushwork Panel can be cut into any other smooth quartz flushwork (besides Thing) in a Stonecutter.
