Smooth Red Sandstone Flushwork (Knapped)

<>Smooth red sandstone flushwork are a set of smooth red sandstone (Minecraft)-based block (Minecraft) in the Minecraft mod (Minecraft) Knapped (Minecraft mod) that can be interchanged between each other in a Stonecutter (Minecraft). The only smooth red sandstone flushwork not included within this set is for the Smooth Red Sandstone Flushwork Thing (Knapped), which requires an enchanted golden apple (Minecraft). The set includes the Smooth Red Sandstone Flushwork Bar (Knapped), Smooth Red Sandstone Flushwork Body (Knapped), Smooth Red Sandstone Flushwork Checkerboard (Knapped), Smooth Red Sandstone Flushwork Cross (Knapped), Smooth Red Sandstone Flushwork Crowd (Knapped), Smooth Red Sandstone Flushwork Diamonds (Knapped), Smooth Red Sandstone Flushwork Flower (Knapped), Smooth Red Sandstone Flushwork Fortress (Knapped), Smooth Red Sandstone Flushwork Panel (Knapped), Smooth Red Sandstone Flushwork Point (Knapped), Smooth Red Sandstone Flushwork Quarters (Knapped), and Smooth Red Sandstone Flushwork Ring (Knapped). The only crafting (Minecraft) flushwork is the Smooth Red Sandstone Flushwork Panel (Knapped), using 1 smooth red sandstone (Minecraft) and 2 knapped flint brick (Knapped) (shapeless). All types of smooth red sandstone flushwork (besides Smooth Red Sandstone Flushwork Thing (Knapped)) can be cut in a Stonecutter (Minecraft) to make the other types of flushwork.
