Smooth Red Sandstone Flushwork Checkerboard (Knapped)

The smooth red sandstone flushwork Checkerboard is a smooth red sandstone flushwork block in the Minecraft mod Knapped. The smooth red sandstone flushwork Checkerboard can be cut from any other smooth red sandstone flushwork (besides Thing) in a Stonecutter.
