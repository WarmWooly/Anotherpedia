Smooth Red Sandstone Flushwork Thing (Knapped)

The smooth red sandstone flushwork Thing is a decorative block in the Minecraft mod Knapped. The smooth red sandstone flushwork Thing can be crafted with 1 smooth red sandstone, 2 knapped flint bricks (item), and 1 enchanted golden apple (shapeless).
