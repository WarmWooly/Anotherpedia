Smooth Sandstone Flushwork (Knapped)

<>Smooth sandstone flushwork are a set of smooth sandstone (Minecraft)-based block (Minecraft) in the Minecraft mod (Minecraft) Knapped (Minecraft mod) that can be interchanged between each other in a Stonecutter (Minecraft). The only smooth sandstone flushwork not included within this set is for the Smooth Sandstone Flushwork Thing (Knapped), which requires an enchanted golden apple (Minecraft). The only crafting (Minecraft) flushwork is the Smooth Sandstone Flushwork Panel (Knapped), using 1 smooth sandstone (Minecraft) and 2 knapped flint brick (Knapped) (shapeless). All types of smooth sandstone flushwork (besides Smooth Sandstone Flushwork Thing (Knapped)) can be cut in a Stonecutter (Minecraft) to make the other types of flushwork.
