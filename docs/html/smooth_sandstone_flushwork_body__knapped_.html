Smooth Sandstone Flushwork Body (Knapped)

The smooth sandstone flushwork Body is a smooth sandstone flushwork block in the Minecraft mod Knapped. The smooth sandstone flushwork Body can be cut from any other smooth sandstone flushwork (besides Thing) in a Stonecutter. The smooth sandstone flushwork Body can be cut into any other smooth sandstone flushwork (besides Thing) in a Stonecutter.
