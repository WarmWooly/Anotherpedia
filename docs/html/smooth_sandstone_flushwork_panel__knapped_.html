Smooth Sandstone Flushwork Panel (Knapped)

<>The smooth sandstone flushwork Panel is a smooth sandstone flushwork block in the Minecraft mod Knapped. The smooth sandstone flushwork Panel can be crafted with 1 smooth sandstone and 2 knapped flint bricks (item) (shapeless). The smooth sandstone flushwork Panel can be cut from any other smooth sandstone flushwork (besides Thing) in a Stonecutter. The smooth sandstone flushwork Panel can be cut into any other smooth sandstone flushwork (besides Thing) in a Stonecutter.
