Smooth Sandstone Flushwork Thing (Knapped)

The smooth sandstone flushwork Thing is a decorative block in the Minecraft mod Knapped. The smooth sandstone flushwork Thing can be crafted with 1 smooth sandstone, 2 knapped flint bricks (item), and 1 enchanted golden apple (shapeless).
