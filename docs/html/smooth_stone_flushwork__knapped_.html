Smooth Stone Flushwork (Knapped)

<>Smooth stone flushwork are a set of smooth stone (Minecraft)-based block (Minecraft) in the Minecraft mod (Minecraft) Knapped (Minecraft mod) that can be interchanged between each other in a Stonecutter (Minecraft). The only smooth stone flushwork not included within this set is for the Smooth Stone Flushwork Thing (Knapped), which requires an enchanted golden apple (Minecraft). The set includes the Smooth Stone Flushwork Bar (Knapped), Smooth Stone Flushwork Body (Knapped), Smooth Stone Flushwork Checkerboard (Knapped), Smooth Stone Flushwork Cross (Knapped), Smooth Stone Flushwork Crowd (Knapped), Smooth Stone Flushwork Diamonds (Knapped), Smooth Stone Flushwork Flower (Knapped), Smooth Stone Flushwork Fortress (Knapped), Smooth Stone Flushwork Panel (Knapped), Smooth Stone Flushwork Point (Knapped), Smooth Stone Flushwork Quarters (Knapped), and Smooth Stone Flushwork Ring (Knapped). The only crafting (Minecraft) flushwork is the Smooth Stone Flushwork Panel (Knapped), using 1 smooth stone (Minecraft) and 2 knapped flint brick (Knapped) (shapeless). All types of smooth stone flushwork (besides Smooth Stone Flushwork Thing (Knapped)) can be cut in a Stonecutter (Minecraft) to make the other types of flushwork.
