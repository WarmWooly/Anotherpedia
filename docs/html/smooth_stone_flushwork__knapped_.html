Smooth Stone Flushwork (Knapped)

<resistance|Noinfo>>Smooth stone flushwork are a set of smooth stone-based blocks in the Minecraft mod Knapped that can be interchanged between each other in a Stonecutter. The only smooth stone flushwork not included within this set is for the Thing, which requires an enchanted golden apple. The set includes the Bar, Body, Checkerboard, Cross, Crowd, Diamonds, Flower, Fortress, Panel, Point, Quarters, and Ring. The only craftable flushwork is the Panel, using 1 smooth stone and 2 knapped flint bricks (item) (shapeless). All types of smooth stone flushwork (besides Thing) can be cut in a Stonecutter to make the other types of flushwork.
