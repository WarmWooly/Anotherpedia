Spinning object meme

A spinning object meme is a internet meme of a person, character, or object spinning. Spinning object memes utilize juxtaposition between the animation of the object spinning and what is being spun. Spinning object memes may be shared as GIF or as video with accompanying music to add to the humor and absurdity of the meme. The rotation speed of the object also greatly impacts how the meme is received, with a faster spin being more chaotic than a slower spin. While spinning object memes primarily utilize 3D model, a 2D animation can also be used to create the illusion of a spinning object.
