Starting Weapons (Laser Fight)

Starting Weapons is a mode (Laser Fight) in Laser Fight (Roblox game) where the player spawns in with three random gun (Laser Fight). This is similar to the mode One Gun Run (Laser Fight), where players spawn with 1 random gun instead of 3. Guns will not gun spawning (Laser Fight) and can't be dropped in this mode.
