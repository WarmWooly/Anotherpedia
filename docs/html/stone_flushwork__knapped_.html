Stone Flushwork (Knapped)

<>{{bStone flushwork}} are a set of stone (Minecraft)-based block (Minecraft) in the {{iMinecraft}} mod (Minecraft) {{iKnapped (Minecraft mod)}} that can be interchanged between each other in a Stonecutter (Minecraft). The only stone flushwork not included within this set is for the Stone Flushwork Thing (Knapped), which requires an enchanted golden apple (Minecraft). The set includes the Stone Flushwork Bar (Knapped), Stone Flushwork Body (Knapped), Stone Flushwork Checkerboard (Knapped), Stone Flushwork Cross (Knapped), Stone Flushwork Crowd (Knapped), Stone Flushwork Diamonds (Knapped), Stone Flushwork Flower (Knapped), Stone Flushwork Fortress (Knapped), Stone Flushwork Panel (Knapped), Stone Flushwork Point (Knapped), Stone Flushwork Quarters (Knapped), and Stone Flushwork Ring (Knapped). The only crafting (Minecraft) flushwork is the Stone Flushwork Panel (Knapped), using 1 stone (Minecraft) and 2 knapped flint brick (Knapped) (shapeless). All types of stone flushwork (besides Stone Flushwork Thing (Knapped)) can be cut in a Stonecutter (Minecraft) to make the other types of flushwork.
