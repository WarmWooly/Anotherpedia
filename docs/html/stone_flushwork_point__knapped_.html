Stone Flushwork Point (Knapped)

<>The stone flushwork Point is a stone flushwork block in the Minecraft mod Knapped. The stone flushwork Point can be cut from any other stone flushwork (besides Thing) in a Stonecutter. The stone flushwork Point can be cut into any other stone flushwork (besides Thing) in a Stonecutter.
