Strengthened Leather Padding (Not Lazy Chainmail)

<>Strengthened leather padding is an item in the Minecraft mod Not Lazy Chainmail as a direct upgrade to the leather padding. Strengthened leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving +0.5 Armor. Strengthened leather padding can be melded in a Melder using 1 leather padding and 1 iron composite ingot. Master-level leatherworkers have a 50% chance to trade 1 leather padding and 16 emeralds for 1 strengthened leather padding. In a leathersmithing table, strengthened leather padding can be applied to chainmail armor or shields. Each strengthened leather padding adds +0.5 Armor, with a full set of armor with 12 strengthened leather paddings giving +6 Armor. Shields will only add Armor when held in the main hand or offhand.
