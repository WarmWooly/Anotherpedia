Swift Leather Padding (Not Lazy Chainmail)

<>Swift leather padding is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the leather padding (Not Lazy Chainmail). Swift leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +0.05 Movement Speed (Minecraft). Swift leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 1 block of gold chains (Not Lazy Chainmail). In a leathersmithing table (Not Lazy Chainmail), swift leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each swift leather padding adds +0.05 Movement Speed, with a full set of armor with 12 swift leather paddings giving +0.6 Movement Speed. Shields will only add Movement Speed when held in the main hand (Minecraft) or offhand (Minecraft).
