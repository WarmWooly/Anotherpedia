Swift Leather Padding (Not Lazy Chainmail)

<>Swift leather padding is an item in the Minecraft mod Not Lazy Chainmail as an upgrade to the leather padding. Swift leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving +0.05 Movement Speed. Swift leather padding can be melded in a Melder using 1 leather padding and 1 block of gold chains. In a leathersmithing table, swift leather padding can be applied to chainmail armor or shields. Each swift leather padding adds +0.05 Movement Speed, with a full set of armor with 12 swift leather paddings giving +0.6 Movement Speed. Shields will only add Movement Speed when held in the main hand or offhand.
