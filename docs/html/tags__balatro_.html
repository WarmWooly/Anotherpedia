Tags (Balatro)

Tags are an optional benefit when skipping a Small Blind (Balatro) or Big Blind (Balatro) in Balatro. There are a total of 24 Tags that can be obtained by skipping a Small or Big Blind. Tags can give Booster Packs (Balatro), create or add an addition to a Editions (Balatro) in The Shop (Balatro), award money (Balatro), increase hand size (Balatro), add Vouchers (Balatro), and other beneficial effects. Selling (Balatro) the Joker Diet Cola (Balatro) will give a Double Tag (Balatro) and defeating a Boss Blind (Balatro) with the Anaglyph Deck (Balatro) will also give a Double Tag.<>
