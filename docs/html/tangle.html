Tangle

For other terms relating to tangle, see tangle (disambiguation).
A tangle is an object, often long and thin like a piece of twine or string, twisted and tied onto itself in a chaotic way. When something is in this state, the object is said to be tangled. Multiple things can be tangled together. A tangle can also be used to describe a conflict between two or more people.
