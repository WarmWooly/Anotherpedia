Technically correct

<>{{tFor the [[meme|internet meme]] from {{ai[[Futurama]]}}, see [[You are technically correct, the best kind of correct]].}}&sp<>{{bTechnically correct}} is a term used to describe when a [[statement]] is [[correct]], but in a way that is [[unusual]] or [[unexpected]]. For example, while "finding" words in a [[word search]] by circling the words in the key is technically correct, as the person found the words on the page, it is not how a word search would normally be solved.
