Telebelt (Not Lazy Chainmail)

<>The telebelt is a belt curios (Not Lazy Chainmail) in the Minecraft mod (Minecraft) Not Lazy Chainmail that allows the wearer to teleport to 3 set locations instantly. Telebelts can be melded in a Melder (Not Lazy Chainmail) using 1 iron composite chainmail belt (Not Lazy Chainmail) and 2 telemagnetic alloy (Not Lazy Chainmail). When the telebelt is worn in a belt slot, the user can press a keybind (Minecraft) (codeK by default) to open up the belt's teleport menu. The user can save a location in the Overworld (Minecraft), Nether (Minecraft), or End (Minecraft) and teleport to those locations using the "Save" and "Teleport" buttons, respectively. Teleporting makes a fwoosh sound at the start and end points of the teleport. By default, teleports are set to 0, 100, 0 XYZ in the Overworld. Other modded dimensions will not work, defaulting to the Overworld.
