Telebelt (Not Lazy Chainmail)

The telebelt is a belt curio in the Minecraft mod Not Lazy Chainmail that allows the wearer to teleport to 3 set locations instantly. Telebelts can be melded in a Melder using 1 iron composite chainmail belt and 2 telemagnetic alloy. When the telebelt is worn in a belt slot, the user can press a keybind (K by default) to open up the belt's teleport menu. The user can save a location in the Overworld, Nether, or End and teleport to those locations using the "Save" and "Teleport" buttons, respectively. Teleporting makes a fwoosh sound at the start and end points of the teleport. By default, teleports are set to 0, 100, 0 XYZ in the Overworld. Other modded dimensions will not work, defaulting to the Overworld. Telebelts can be crushed into unfinished iron composite chainmail projects containing 128 chain links.
