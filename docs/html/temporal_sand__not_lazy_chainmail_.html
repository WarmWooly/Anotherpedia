Temporal Sand (Not Lazy Chainmail)

Temporal sand is a gravity-affected block in the Minecraft mod Not Lazy Chainmail. Temporal sand can be obtained when a Crusher crushes end stone. A Cobblestone Generator with an end'n'crushifier will produce temporal sand. End'n'crushifiers take 64x as long to generate one temporal sand compared to one cobblestone. Temporal sand can be smelted in a furnace to produce temporal glass, taking 10 seconds and giving 1 experience. Temporal sand can be sieved, producing various End-based items.
