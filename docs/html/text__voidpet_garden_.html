Text (Voidpet Garden)

Text is an action that a user can partake in Voidpet Garden where they text messaging an Generative artificial intelligence chatbot masking as their Voidpet (fictional species). Users can send multiple messages to their Voidpet, with it responding in turn, influenced by their emotion (i.e. Joy (Voidpet) being joyful). The number of daily messages is limited for free users, with the cap raised for subscription to Voidpet Giga. Message exchanges do not get saved without an attached account.
