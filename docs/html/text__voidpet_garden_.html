Text (Voidpet Garden)

Text is an action that a user can partake in Voidpet Garden where they message an AI chatbot masking as their Voidpet. Users can send multiple messages to their Voidpet, with it responding in turn, influenced by their emotion (i.e. Joy being joyful). The number of daily messages is limited for free users, with the cap raised for subscribers to Voidpet Giga. Message exchanges do not get saved without an attached account.
