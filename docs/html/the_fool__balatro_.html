The Fool (Balatro)

The Fool (0) is a Tarot Card in Balatro based on The Fool Major Arcana tarot card. When used, it converts itself into the last Tarot or Planet Card used. It can't be used if the last Tarot Card was The Fool. If all Tarot Cards are held while generating a random Tarot Card, it will select The Fool (ex. using Perkeo to create a Negative copy of every Tarot Card will allow this to happen).
