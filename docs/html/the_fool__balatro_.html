The Fool (Balatro)

{{bThe Fool (0)}} is a Tarot Card (Balatro) in {{iBalatro}} based on The Fool (tarot card) Major Arcana tarot card. When used, it converts itself into the last Tarot or Planet Cards (Balatro) Card used. It can't be used if the last Tarot Card was The Fool. If all Tarot Cards are held while generating a random Tarot Card, it will select The Fool (ex. using Perkeo (Balatro) to create a Negative (Balatro) copy of every Tarot Card will allow this to happen).
