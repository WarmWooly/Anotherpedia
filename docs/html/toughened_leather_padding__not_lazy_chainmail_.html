Toughened Leather Padding (Not Lazy Chainmail)

<>Toughened leather padding is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the leather padding (Not Lazy Chainmail). Toughened leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +1 Armor Toughness (Minecraft). Toughened leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 1 diamond (Minecraft). Expert-level leatherworker (Minecraft) have a 50% chance to trade 1 leather padding (Not Lazy Chainmail) and 6 emerald (Minecraft) for 1 toughened leather padding. In a leathersmithing table (Not Lazy Chainmail), toughened leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each toughened leather padding adds +1 Armor Toughness (Minecraft), with a full set of armor with 12 toughened leather paddings giving +12 Armor Toughness. Shields will only add Armor Toughness when held in the main hand (Minecraft) or offhand (Minecraft). Toughened leather paddings are used in the following melding recipes:
Draconic Leather Padding (Not Lazy Chainmail) - 1 toughened leather padding, 1 inflated leather padding (Not Lazy Chainmail), and 1 dragon head (Minecraft)
