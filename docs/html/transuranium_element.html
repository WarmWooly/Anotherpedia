Transuranium element

A transuranium element is a term used to refer to elements with a greater atomic number than uranium (92). All these elements undergo radioactive decay and (besides neptunium and plutonium in rare cases) are synthetic elements. <>, there are only 26 transuranium elements, 15 of those being superheavy elements (past 103). The transuranium elements (in order of atomic number) are neptunium (93), plutonium (94), americium (95), curium (96), berkelium (97), californium (98), einsteinium (99), fermium (100), mendelevium (101), nobelium (102), lawrencium (103), rutherfordium (104), dubnium (105), seaborgium (106), bohrium (107), hassium (108), meitnerium (109), darmstadtium (110), roentgenium (111), copernicium (112), nihonium (113), flerovium (114), moscovium (115), livermorium (116), tennessine (117), and oganesson (118).
