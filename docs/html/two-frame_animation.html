Two-frame animation

A two-frame animation is a simple animation made up of only two frames. Two-frame animations come in many forms and can be used to show the basics of animation. For example, a two-frame animation can have one frame at rest with the second having an action being performed (the flapping of a wing; the waving of a hand). It can also be done to create a consistent, small movement in an otherwise still figure. The frame rate greatly impacts the two-frame animation as there are only two frames to change between. Two-frame animation is especially popular for basic stop motion animations.
