Uncharacteristically

Uncharacteristically is an adverb used to describe when an object or person acts or behaves in a way that is unexpected, as it is not a normal or usual characteristic exhibited by the object or person. Uncharacteristically is comprised of the prefix un-, the adjective characteristic (which itself is a combination of the noun character and the suffix -isitc), and the suffix -ally.
