Uncharacteristically

{{bUncharacteristically}} is an adverb used to describe when an object or person action or behavior in a way that is unexpected, as it is not a normal or usual characteristic exhibited by the object or person. Uncharacteristically is comprised of the prefix {{iun-}}, the adjective {{icharacteristic}} (which itself is a combination of the noun {{icharacter}} and the suffix {{i-isitc}}), and the suffix {{i-ally}}.
