Uncharacteristically

<><>{{bUncharacteristically}} is an [[adverb]] used to describe when an [[object]] or [[person]] [[acts|action]] or [[behaves|behavior]] in a way that is [[unexpected]], as it is not a [[normal]] or [[usual]] [[characteristic]] exhibited by the object or person. Uncharacteristically is comprised of the [[prefix]] {{i[[un-]]}}, the [[adjective]] {{icharacteristic}} (which itself is a combination of the [[noun]] {{i[[character]]}} and the [[suffix]] {{i[[-isitc]]}}), and the suffix {{i[[-ally]]}}.
