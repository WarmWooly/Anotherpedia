Uncyclopedia

{{tFor the big brain version of Uncyclopedia, see Wikipedia.}}&sp{{bUncyclopedia}} is a satire online encyclopedia, parody Wikipedia, which can be found at <>. Its name is a combination of the prefix {{iun-}} and {{iencyclopedia}}. Its logo is a parody of Wikipedia's, being a hollowed potato instead of a hollowed sphere. Uncyclopedia was created in 2005 by Jonathan Huang, where user, known as Uncyclopedian, are able to create and edit articles.
