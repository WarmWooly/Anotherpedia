Unfinished Copper Project (Not Lazy Chainmail)

The unfinished copper project is an item in the Minecraft mod Not Lazy Chainmail used to be snipped into copper chain links. In a chainmail project with at least 1 copper chain link added, it can be reset to return all the used chain links in the form of an unfinished copper project. Unfinished copper projects can be obtained when a Crusher crushes the following items (storing the listed number of chain links):
Copper Chainmail Helmet - 320 chain links
Copper Chainmail Chestplate - 512 chain links
Copper Chainmail Leggings - 448 chain links
Copper Chainmail Boots - 256 chain links
Copper Chainmail Bracelet - 64 chain links
Copper Chainmail Necklace - 128 chain links
Copper Chainmail Gloves - 192 chain links
Copper Chainmail Belt - 128 chain links
Copper Tool Chain - 192 chain links
Copper Chainmail Industrial Belt - 256 chain links
Music Disc "Bounce" - 32 chain links Pliers can be used to snip the unfinished project into copper chain links. The number of chain links in the unfinished project is equal to the number put into the chainmail project before it is reset or the total required for the crushed item. gal
