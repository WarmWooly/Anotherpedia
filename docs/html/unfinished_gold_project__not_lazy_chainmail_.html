Unfinished Gold Project (Not Lazy Chainmail)

The unfinished gold project is an item in the Minecraft mod Not Lazy Chainmail used to be snipped into gold chain links. In a chainmail project with at least 1 gold chain link added, it can be reset to return all the used chain links in the form of an unfinished gold project. Unfinished gold projects can be obtained when a Crusher crushes the following items (storing the listed number of chain links):
Gold Chainmail Helmet - 320 chain links
Gold Chainmail Chestplate - 512 chain links
Gold Chainmail Leggings - 448 chain links
Gold Chainmail Boots - 256 chain links
Gold Chainmail Bracelet - 64 chain links
Gold Chainmail Necklace - 128 chain links
Gold Chainmail Gloves - 192 chain links
Gold Chainmail Belt - 128 chain links
Gold Tool Chain - 192 chain links
Gold Chainmail Industrial Belt - 256 chain links
Music Disc "Bounce" - 32 chain links Pliers can be used to snip the unfinished project into gold chain links. The number of chain links in the unfinished project is equal to the number put into the chainmail project before it is reset or the total required for the crushed item. gal
