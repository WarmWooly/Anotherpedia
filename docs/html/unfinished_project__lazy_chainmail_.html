Unfinished Project (Lazy Chainmail)

Not to be confused with Unfinished Iron Project ({{aiNot Lazy Chainmail).}}
The unfinished project is an item in the Minecraft mod Lazy Chainmail used to be snipped into iron chain links. In a chainmail project with at least 1 iron chain link added, it can be reset to return all the used chain links in the form of an unfinished project. Pliers can be used to snip the unfinished project into iron chain links. The number of chain links in the unfinished project is equal to the number put into the chainmail project before it is reset.
