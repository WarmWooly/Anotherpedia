Vein

<>{{tFor the [[fracture]] containing [[ores|ore]], see [[vein (geology)]].}}&sp{{tNot to be confused with [[vain]] or [[vane]].}}&sp<>A {{bvein}} is a [[blood vessel]] in a [[body]] (typically a [[human's|human]]) part of the [[circulatory system]] used for carrying [[blood]]. Veins are used to describe large blood vessels which carry blood in need of [[oxygen]] towards the [[lungs|lung]], which then feeds into [[arteries|artery]] to carry the oxygenated blood around the body.
