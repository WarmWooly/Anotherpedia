Vein

{{tFor the fracture containing ore, see vein (geology).}}&sp{{tNot to be confused with vain or vane.}}&spA {{bvein}} is a blood vessel in a body (typically a human) part of the circulatory system used for carrying blood. Veins are used to describe large blood vessels which carry blood in need of oxygen towards the lung, which then feeds into artery to carry the oxygenated blood around the body.
