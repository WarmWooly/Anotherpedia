Vein

For the fracture containing ores, see vein (geology).&spNot to be confused with vain or vane.&spA vein is a blood vessel in a body (typically a human's) part of the circulatory system used for carrying blood. Veins are used to describe large blood vessels which carry blood in need of oxygen towards the lungs, which then feeds into arteries to carry the oxygenated blood around the body.
