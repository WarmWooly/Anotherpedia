Vitriol

For the generalized term for sulfates, see vitriol (chemistry), and for the band, see Vitriol (band).&spVitriol is a noun used to describe a cruel or aggravated anger or other negative emotion, mostly used in the context of what someone is saying. If someone is saying a statement with vitriol, it is bitter and resentful.
