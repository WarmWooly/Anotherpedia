Vitriol

For the generalized term for sulfate, see vitriol (chemistry), and for the band, see Vitriol (band).
Vitriol is a noun used to describe a cruel or aggravation anger or other negative emotion, mostly used in the context of what someone is saying. If someone is saying a statement with vitriol, it is bitter and resentful.
