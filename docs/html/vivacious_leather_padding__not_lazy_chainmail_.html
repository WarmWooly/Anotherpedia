Vivacious Leather Padding (Not Lazy Chainmail)

<>Vivacious leather padding is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the healthy leather padding (Not Lazy Chainmail). Vivacious leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +6 Max Health (Minecraft) (3 hearts) and +0.02 Gravity (Forge). Vivacious leather padding can be melded in a Melder (Not Lazy Chainmail) using 2 healthy leather padding (Not Lazy Chainmail) and 1 block of rose gold chains (Not Lazy Chainmail). In a leathersmithing table (Not Lazy Chainmail), vivacious leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each vivacious leather padding adds +6 Max Health and +0.02 Gravity, with a full set of armor with 12 vivacious leather paddings giving +72 Max Health (36 hearts) and +0.24 Gravity. Shields will only add Max Health and Gravity when held in the main hand (Minecraft) or offhand (Minecraft).
