Vouchers (Balatro)

{{bVouchers}} are a permanent upgrade that can be purchased from The Shop (Balatro) during a run (Balatro) in {{iBalatro}}. There are a total of 32 Vouchers that can be purchased in a run. Vouchers have two tiers, with the second tier being unlocked after the first tier of a Voucher is bought. Only one of each Voucher may be bought. There is only 1 Voucher available per Shop that only restocks after the Boss Blind (Balatro). A Voucher costs Money (Balatro)10. Using the Voucher Tag will create another Voucher in the Shop.<>
