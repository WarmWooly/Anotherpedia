Website

A website is a collection of related web page connected under a single web domain that is stored on a web server. hyperlink are used to navigate between pages on a website, which also typically include some form of home page. All websites make up the World Wide Web. Most websites have one to two specific purpose, such as the news, the weather, online shopping, hosting and sharing online video platform, social media platform, and more.
