Weighted Leather Padding (Not Lazy Chainmail)

<>Weighted leather padding is an item (Minecraft) in the Minecraft mod (Minecraft) Not Lazy Chainmail as an upgrade to the leather padding (Not Lazy Chainmail). Weighted leather padding is used to upgrade chainmail armor (Not Lazy Chainmail) or shield (Minecraft) in a leathersmithing table (Not Lazy Chainmail), giving +0.02 Gravity (Forge). Weighted leather padding can be melded in a Melder (Not Lazy Chainmail) using 1 leather padding (Not Lazy Chainmail) and 1 iron ingot (Minecraft). In a leathersmithing table (Not Lazy Chainmail), weighted leather padding can be applied to chainmail armor (Not Lazy Chainmail) or shield (Minecraft). Each weighted leather padding adds +0.02 Gravity, with a full set of armor with 12 weighted leather paddings giving +0.24 Gravity. Shields will only add Gravity when held in the main hand (Minecraft) or offhand (Minecraft).
