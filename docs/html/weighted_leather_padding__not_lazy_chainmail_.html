Weighted Leather Padding (Not Lazy Chainmail)

Weighted leather padding is an item in the Minecraft mod Not Lazy Chainmail as an upgrade to the leather padding. Weighted leather padding is used to upgrade chainmail armor or shields in a leathersmithing table, giving +0.02 Gravity. Weighted leather padding can be melded in a Melder using 1 leather padding and 1 iron ingot. In a leathersmithing table, weighted leather padding can be applied to chainmail armor or shields. Each weighted leather padding adds +0.02 Gravity, with a full set of armor with 12 weighted leather paddings giving +0.24 Gravity. Shields will only add Gravity when held in the main hand or offhand.
