When to use an 'An' before an abbreviation

Each abbreviation that comes after the word a a must also follow the rules of a and an because each letter has a name, which may start with letters with nouns, thus turning an 'a' into an 'an'. This is a quick guide on when to use such rules for abbreviations. This only applies for the first letter of an abbreviation (such as "This is an HDD"), even when the unabbreviated form uses a different word (such as "This is a hard disk drive").

B (Bee) = a
C (Cee) = a
D (Dee) = a
E (E) = an
F (Ef) = an
G (Gee) = a
H (Aitch) = an
I (I) = an
J (Jay) = a
K (Kay) = a
L (El or Ell) = an
M (Em) = an
N (En) = an

P (Pee) = a
Q (Cue) = a

S (Ess) = an
T (Tee) = a
