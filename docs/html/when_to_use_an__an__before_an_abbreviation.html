When to use an 'An' before an abbreviation

Each abbreviation that comes after the word A (word) a must also follow the Rule of a and an because each letter has a Alphabet Names, which may start with letters with nouns, thus turning an 'a' into an 'an'. This is a quick guide on when to use such rules for abbreviations.
