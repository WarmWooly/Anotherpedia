Whiplash with Quick Time Events

"Whiplash with Quick Time Events (keyboard needed) #ytqte" is a YouTube quick time events posted by BagWorksCam on December 10, 2025 of the 2014 film Whiplash (2014 film). The player for the video is Andrew Neiman, with the main antagonist being Terence Fletcher. The player must use various keybind that YouTube uses to skip around the video, creating the illusion of an interactive game by using snippets of the video. A Whiplash with Quick Time Events 2 would be released the following January.
