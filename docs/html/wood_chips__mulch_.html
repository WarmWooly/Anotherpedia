Wood chips (Mulch)

<><>||{{bStats}}||{{b[[Rarity|Rarity (Minecraft)]]}}|Common||{{b[[Stack size|Stack size (Minecraft)]]}}|64||{{b[[Fire/lava&spresistance|Fire-resistant item (Minecraft)]]}}|Noinfo>>{{bWood chips}} are a type of [[item|item (Minecraft)]] in the {{i[[Minecraft]]}} [[mod|mod (Minecraft)]] {{i[[Mulch|Mulch (Minecraft mod)]]}} used to [[compost|composter (Minecraft)]] and to [[craft|crafting (Minecraft)]] [[mulch|mulch (Mulch)]].<><>{{tFor additional recipes added for other mods, see [[{{aiMulch}} mod compatibilities]].}}&spWood chips can only be obtained through chipping wood in a [[wood chipper|wood chipper (Mulch)]]. The following are all the blocks and items that can be chipped into wood chips:<><><>4 wood chips of the same type in a 2x2 square can be used to [[craft|crafting (Minecraft)]] [[mulch|mulch (Mulch)]] of the same type. For example, 4 oak wood chips would make 1 oak mulch. 6 of any wood chips can also be crafted into [[wood chip mix|wood chip mix (Mulch)]] (shapeless). 2 wood chips of the same time in a 2-wide horizontal line can be used to craft [[mulch slabs|mulch slabs (Mulch)]] of the same type.<>All wood chips, besides Crimson, Warped, and Hellfire, can be used as [[fuel|fuel (Minecraft)]] in a [[furnace|furnace (Minecraft)]], smelting 0.75 items per item.<>Placing wood chips into a [[composter|composter (Minecraft)]] has an 85% chance of raising the compost level by 1. On average, it takes 8.06 wood chips to fill a composter, with every [[stack|stack (Minecraft)]] averaging 7.94 [[bonemeal|bonemeal (Minecraft)]] composted.<>All wood chips can be thrown into a [[mulch redefiner|mulch redefiner (mulch)]] to be turned into all non-dyed wood chips.
