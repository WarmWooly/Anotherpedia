The wooden pickaxe deals 2 damage and has an attack cooldown of 1.2 (0.85 seconds or 17 [[ticks|tick (Minecraft)]]) on [[Java|Minecraft: Java Edition]] and deals 3 damage on [[Bedrock|Minecraft: Bedrock Edition]].<>The wooden pickaxe can be repaired through [[combining|combining (Minecraft)]] by [[crafting|crafting (Minecraft)]], using a [[grindstone|grindstone (Minecraft)]], or using an [[anvil|anvil (Minecraft)]]. Combining will remove all enchantments, with some [[experience|experience (Minecraft)]] returned when combining in a grindstone.
